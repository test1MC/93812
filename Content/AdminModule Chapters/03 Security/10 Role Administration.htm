Role Administration.

 

Edit Mode: Grid or form.

Note: While you may edit data directly in the grid, it's easier to edit data via the Role Administration form.

Description: The Role Administration option allows Administrators to modify some role information. Ideally, all necessary roles are created when the application is initially developed.

Note: Changes may be needed throughout the System Administration module to enable the new role to become effective. Special access rules may also require customization of the application, which is not available through System Administration. If you have any questions about this, please contact Libera.

Refer to the Role Administration Form topic for more detailed information on working with the Role Administration form

 

Adding a New Role
  1. Select the Action Row where it says, "Please click here to add a new record..." This opens a new Role Administration form.
  2. Fill out the form as you would any other form.

 

 Modifying an Existing Role
  1. Select the Role in the grid to open its Role Administration form.
  2. Edit fields on the form as you would with any other form.
