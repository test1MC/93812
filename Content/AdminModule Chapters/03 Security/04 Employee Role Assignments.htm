Employee Role Assignments.

 

Edit Mode: Grid

Description: The Employee Role Assignments option shows all the roles assigned to each employee. It serves as a quick and effective way to manage user roles. The grid is grouped by employee and shows the employee’s last name, first name, social security number, office name, office address, and office phone number. When you click on the “+” sign next to the employee’s name or press the spacebar while in the row containing the employee’s name, this allows you to see the roles assigned to that employee and whether or not that role is considered primary.

Administrators can update the employee’s roles whenever the employee’s job responsibilities change. Role information can either be changed in this grid or via the Employee Information form accessible via the Employee Administration option.

 

Adding a New Role
  1. Select the Action Row where it says, "Please click here to add a new record..." This opens the row for editing. The Role ID and Role Name cells have lookups. The Primary Role? cell is a check box.
  2. Make selections for each cell, then press CTRL+ENTER or single-click into the grid to save your changes.

 

 Modifying an Existing Role
  1. Open (right-click or press F4) the cell context menu in the cell that you will be changing. Select the Edit option. This opens the cell up for editing.
  2. Edit the cells the same as if you were adding a new role.
  3. Press CTRL+ENTER or single-click into another row in the grid to save your changes.
