Dictionary Administration



Dictionary Administration.

 

Edit Mode: Grid

Description: Dictionary Administration allows Administrators to manage the dictionaries (editable lists) used for form lookup lists. These lookup lists are located in forms. When the user opens the lookup, the lookup is populated from a dictionary table.

The grid in the ECF lists all dictionaries with identifying information about each dictionary. Select an option in the ECF to open the dictionary. This opens a new window displaying the contents of the dictionary. Administrators can change the contents of dictionaries.

Note: Be careful when modifying dictionaries. If an option is deleted that was previously selected (at least once), unpredictable results may occur. Also, it is important to be careful about modifying dictionaries that contain information which is used in federal reports. The federal reports pull certain codes that are used in some of these dictionary tables. If federal codes or the meaning of the codes are changed so that they no longer match the federal report instructions this will affect the accuracy of the federal reports.

 



Example of a Dictionary Administration table.

 

The contents of each dictionary table contains different columns and rows. Some have only a few of each, others have many of each.

 

Adding a New Record to a Dictionary
  1. Select the Action Row where it says, "Please click here to add a new record..." This action opens the row for editing.
  2. Type directly into the cells.
  3. When you're done, press CTRL+ENTER or single-click into the grid to save your data entry.


    Dictionary table while adding a new record and after.

 

 Editing a Dictionary Table Entry
  1. Select the record to be edited, then press F4 or right-click to open the context menu.
  2. Select the Edit option from the context menu. This opens the cell for editing. Now you can change the contents of the cell.


    Editing a cell in the grid.

 

 Deleting a Dictionary Table Entry
  1. Select the record to be edited, then press F4 or right-click to open the context menu.
  2. Select the Delete option from the context menu. A confirmation message appears, "Are you sure you want to delete this record?" If so, select the OK button in the message window, otherwise select the Cancel button.
  3. Selecting the OK button closes the confirmation message window and deletes the record from the grid.

Note: There's no "undo" when deleting. Think carefully before deleting a record from a dictionary table.
