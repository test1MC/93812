System Notifications



System Notifications.

 

Edit Mode: Form

Description: The System Notifications option allows Administrators to use automatic email notifications to inform users of particular events or conditions. These notifications keep individuals informed so they can respond to specific situations in a timely manner or provide reminders of particular job duties. These email notifications are different from the System Messages option which automatically displays a bulletin board type of message when the user logs into the system. System messages are not email messages.

The grid shows basic details of the email notifications, including the notification number and title, subject, active status of the notification, notification type, and the PDQ ID number associated with the notification.

Administrators may access existing notifications to enable or disable them, and to change the user roles that can subscribe to and receive notifications. Libera Developers create new notifications.

 

Adding a New System Notification


The top porition of the System Notification Form.

  1. Select the Action Row where it says, "Please click here to add a new record..." This opens the System Notification Form in a new window.
  2. Fill out the form as you would any other form. The fields are defined directly on the form.

 

 Editing an Existing System Notification
  1. Select a notification record in the ECF. This opens the System Notification Form in a new window.
  2. Edit the form as you would any other form. The fields are defined directly on the form.

 

 Deactivating an Existing System Notification
  1. Select a notification record in the ECF. This opens the System Notification Form in a new window.
  2. Scroll to the bottom of the form. Set the Active Notification field to N.

Note: If you have any questions about working with this form please contact Libera Client Services.
