Scheduled Tasks.

Edit Mode: Form

Description: The Scheduled Tasks option allows Administrators to control tasks that are set up to run automatically, such as running a report or performing an edit check. Administrators can change key information such as the start dates, end dates, the time interval, or the activation of existing tasks.

The grid displays tasks grouped by category. Each task shows an ID number, a task description, a start date and an end date for the task. Additionally, the grid shows interval, last run time, next run time, task type, resource, active status, and last error. If you press ENTER or double-click a record to open the Scheduled Task Setup Form, each of the columns in the ECF grid is defined in this form.

Note: Libera Developers manage the details of creating and maintaining Scheduled Tasks, as specific knowledge of the available components within the application is required.
