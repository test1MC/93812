Scheduled Task Setup Form


The top portion of the Scheduled Task Setup Form.

 

Overview

This form is used to setup scheduled tasks for System 7™ to execute automatically. These can also be run manually by using the Run Task Now button.

 

 

Form Fields

The form includes information about most fields directly on the form. As such, not all fields have additional explanation below.

Note: While you may have the ability to edit many fields within this form, it is advised that you do not make changes without guidance from Libera.

  • Task Short Description - The value for the field appears in the Description column of the Scheduled Task Administration table.
  • Task Long Description - Provides additional description of the task.
  • The Task Resource field is only editable when Task Type is not set to Custom Data Commands.
  • The Task Commands field is only editable when Task Type is set to Custom Data Commands.
  • Notify On Success or Notify On Failure - An email notification is generated upon completion of the task and emailed to the addresses entered into these fields.
  • When running the task using the Run Task Now button, the task time out will not be taken into consideration.
  • View Task History - This button opens a new window displaying the results of the Scheduled Task History PDQ for this task. This information will include the time and duration of the task each time it ran, the server it ran on, and the number of errors (if any) that occurred each time.
