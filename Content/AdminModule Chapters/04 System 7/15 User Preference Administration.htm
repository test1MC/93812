User Preference Administration



User Preference Administration.

 

Edit Mode: Grid

Description: The User Preference Administration option allows you to globally set the Default Value for User Preferences. The settings are grouped by basic categories.

Libera Developers to manage all other aspects of User Preferences including adding or removing them.

Note: While you may have the technical ability to modify the data other than the Default Value within the grid it is strongly suggested that you do not. Doing so may result in unexpected results.

If you would like to remove existing options or add new options, please contact your Supervisor or Libera Support.

 

Parent Grid ECF Columns

  • Group Name - Displays the name of the grouping.
  • Group Description - Displays the description of the group.

 

Child Table ECF Columns

  • System Name - The name of the User Preference setting used in the database.
  • Name - The name of the User Preference option.
  • Description - Describes the functionality of the User Preference option.
  • Data Type - The type of response required for the User Preference. System.String requires a Y/N or T/F response. System.Boolean requires a text response. Both are limited by the options in the lookup specific to the User Preference.
  • Default Value - The preset selection for all users. Note: All users have the ability to set their own User Preferences.
  • PDQ Xml ID - The ID used for the lookup that populates the lookup for the User Prefence.
