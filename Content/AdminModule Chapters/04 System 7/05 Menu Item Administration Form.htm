Menu Item Administration Form


Menu Item Administration form.

 

Overview

The Menu Item Administration form is used to manage Menu Items.

 

 

Form Fields

  • Menu - Menu with which the Menu Item is associated. The Menu is selected at the time the Menu Item is added and cannot be changed.
  • Name - The Menu Item's name that appears within the Menu.
  • Description - Text that provides additional information about the Menu Item that appears below the Name in the Menu.
  • Group/Parent - How the Menu Item is categorized within the Menu. The Menu Item will be found grouped under this heading.
  • Roles - User Roles that can see this Menu Item in the Menu. This field accepts a comma separated list of User Roles. When left blank, all users with access to the Menu have access to this Menu Item.
  • Build Roles - Instead of typing a comma separated list of User Roles, you can use this button to create the list. When selected the Build Roles section of the form opens. Selecting the links opens the User Role list in a new window for you to select which roles you want to have access to the Menu Item or not.


    Using the Build Roles button.
  • Save - Button in the Build Roles section. Saves your user role selections and closes the Build Roles section.

 

Note: Excluded roles appear with an exclamation point infront of them.
