Report Administration Form


The top portion of the Report Administration form.

 

Overview

The Report Administration form is used to add and modify reports. Libera manages the vast majority of the report functionality, however Administrators may safely modify a few options.

 

 

Fields You May Safely Modify

  • Report Name - This is the name of the report as it appears in the Report Administration grid and on the Reports tab of the File Drawer.
  • Description - This is a brief description of the report. It's meant to give users an idea of what the report is about before they run it. The Description appears in the Report Administration grid and on the Reports tab of the File Drawer.
  • Report ACL - A comma separated list of User Roles who can see and run the report. Optionally you can use the Build Report ACL button to select User Roles to include or exclude from accessing the report.

 

Fields Managed by Libera

While you may have the technical ability to modify many of the remaining fields on the form, you should not. They are provided for your information only. Each field is described directly on the form. If you have any questions about these fields, please contact your Supervisor or Libera Client Support.
