Module Administration.

 

Description: The Module Administration option allows System Administrators to create the modules that appear on the Application Home Page and to categorize components. The grid shows the ID, code, and name of the module.

Libera recommends that System Administrators do not add or remove any modules. These types of modifications may cause errors within the application. However, the names of modules may be edited as desired without impacting the functionality of the application.

 

ECF Columns

  • ID - System generated unique identification number for the module. Do not edit.
  • Code - A ten character abbreviation for the module name. Do not edit.
  • Name - Module name. Accepts up to 50 characters. Module names are used to categorize data in the system. Names can be edited.
