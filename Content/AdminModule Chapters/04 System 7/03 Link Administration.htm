Link Administration



Link Administration grid.

 

Edit Mode: Grid

Description: The Link Administration option allows Administrators to manage the hyperlinks provided in the application. These are found in the Links tab of the File Drawer. The grid shows the title of the link, the Web address, and a description of the target site. Administrators can add or modify links as desired to provide users with quick access to valuable information and resources.

 

Adding a New Link
  1. Select the Action Row where it says, "Please click here to add a new record..." This opens the row for editing.
  2. Enter the values into the cells.
  3. Press CTRL+ENTER or single-click into any cell in the grid to add the new record.

 

 Modifying an Existing Link
  1. Open (right-click or press F4) the cell context menu in the cell that you will be changing. Select the Edit option. This opens the cell up for editing.
  2. Make your changes.
  3. Press CTRL+ENTER or single-click into any cell in the grid to add the new record.
