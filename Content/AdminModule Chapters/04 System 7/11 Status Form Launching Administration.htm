Status Form Launching Administration.

 

Edit Mode: Form or Grid

Description: The Status Form Launching Administration option allows Libera Developers to set up which forms are launched as a result of a status change.

Note: Making changes to the Status Form Launching Administration grid should be made only by Libera Developers.

 

ECF Columns

  • Program - Program code of the forms involved.
  • Source Status and Target Status - To and From status moves that trigger the launching of the form.
  • Form ID to Launch - Form Template ID (Also Known As Component #) of the form to be launched when the criteria matches.
  • Component Template ID to Launch From - If applicable, the trigger form.
  • Component Type to Launch From - Type code of the Component Template ID to Launch From component. 1 = PDQ, 2 = form.
  • Form Name - Name of the form to be launched.
  • Launching mode - Code for the form launching functionality.
  • Modal - When selected the form that opens retains focus, meaning that users won't be able to work in another window until the new form window closes.
  • Form Height and Form Width - Indicates the size of the window when the form launches.
  • Form Title - Displays what appears in the title bar of the form window. When blank, the component title alone is displayed.
  • Target - Displays if the form is opened in a new window or within the current form.
  • Parameters - User or system define parameters that control any number of things about the form to be launched.
  • Roles - A comma separated list of User Roles that have access to the form being launched. Can be used to launch different forms for different user groups.
  • Enabled - Indicates if this form launching action is enabled in the system or not.

 

ECF Grid Functionality

  • Selecting a record in the grid opens the associated Status Form Launching Settings form in a new window.
