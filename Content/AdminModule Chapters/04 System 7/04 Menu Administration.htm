Menu Administration



Menu Administration.

 

Edit Mode: No Editing

Description: The Menu Administration option allows Administrators to manage menus in different parts of the system such as the Application Home Page and the System Administration Menu PDQ.

 

ECF Columns

  • Menu Item - This is the title of the option as it appears in its associated menu.
  • Roles - Displays a comma separated list of user roles who have access to this option. When blank, all users can access the option.
  • Module - The module with which the Menu Item is associated.
  • PDQ - If applicable, displays the unique ID number of the PDQ associated with the Menu Item. This is managed by Libera.
  • Commands? - Indicates whether or not a command is used in conjunction with the Menu Item. This is managed by Libera.
  • Menu - The menu with which the Menu Item is associated.
  • Group/Parent - How the Menu Item is grouped in the menu.
  • Stage - Controls when the Menu Item is visible to users. This is managed by Libera.

Adding or Deleting a Menu Item

Libera manages this functionality. If you want Menu Items added or deleted, please contact your Supervisor or Libera Client Support.

Modifying a Menu Item

Note: Use caution when modifying Menu Items. If you have any questions about this process, please contact your Supervisor or Libera Client Support.

  1. Select the Menu Item to be modified in the grid. This action opens the Menu Item Administration form.
  2. Make desired changes within the form.
