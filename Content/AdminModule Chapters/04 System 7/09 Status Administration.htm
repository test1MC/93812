Status Administration.

 

Edit Mode: No editing

Description: The Status Administration option allows Administrators to view the transition between the various statuses in each program. Allowable status movements are listed in a grid with the program number and a description of the new status. This defines various business rules as well as providing information for manual status changes. Though the initial status change rules should match the desired rules for the application, it may be necessary to modify these rules as agency procedures change. Such changes should be made carefully, as they may affect how users are able to perform their tasks. This table is managed directly by Libera. Please direct change requests to your Supervisor or Libera Client Support.

 

ECF Columns

  • Current Status - Current status of the case. A record appears in the grid for every Dest Status for the Current Status. For example, if you filter the grid by a Program of your choice and Current Status 02, you will see all possible statuses that a case in the selected Program can go into from status 02.
  • Program - Program which the status is associated. Note: Many programs may use the same status numbering system. When working with statuses make sure you have the correct Program selected.
  • Dest Status - Destination status to which a case can move to from its Current Status.
  • Description - This describes the Dest Status.
  • Allowed Manually - When selected, users are allowed to move a case into this status via the Case Note form.
  • Manual Change ACL - If a status move is Allowed Manually, you will see which groups of users may manually complete the status move. This data is displayed as a comma separated list of User Role IDs. If no roles are specified, then all users may manually complete the status move.
  • Default Description - When a Dest Status works with multiple Current Statuses, this is the description that is used.
  • Undo Fix Function - This is related to Status History Editor functionality and is maintained by Libera.

 

PDQ Results

 

  • You may find it easer to work with this grid if you first filter or group by program, then sort the Current Status column A to Z.
