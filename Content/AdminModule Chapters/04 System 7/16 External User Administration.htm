External User Administration.

 

Edit Mode: No editing

Description: The External User Administration option must be enabled by Libera Developers before it can be utilized. It allows Administrators to provide access to the system to outside users i.e. Vendors, Clients, etc. The External User Administration grid is for informational purposes only. External Users are added via various forms in the system. In order to see the External Users section within the designated forms, users must have access to the section via the External user authentication Administration ACL Application Setting. The External User Administration grid is grouped by User Type.

 

ECF Grid Columns

  • ID - Unique identification number for the user record.
  • Username - Name used to log into the portal.
  • First Name - User's first name.
  • MI - User's middle initial.
  • Last Name - User's last name.
  • Email - User's email address.
  • User Type -

 

ECF Grid Functionality

  • Selecting a record in the grid opens the External User Administration form.
