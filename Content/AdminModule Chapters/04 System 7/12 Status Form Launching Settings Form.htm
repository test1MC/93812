Status Form Launching Settings Form


Status Form Launching Settings form.

 

Overview

The Status Form Launching Settings form is used to modify the settings for status form launching records.

Note: Making changes to the Status Form Launching Settings form should be made only by Libera Developers.

 

 

Form Functionality

All data entry appears in the Status Form Launching Administration grid.

  • Program - Program code of the forms involved.
  • Source Status and Target Status - To and From status moves that trigger the launching of the form.

  • Roles - A comma separated list of User Roles that have access to the form being launched. Can be used to launch different forms for different user groups.

  • Form ID to Launch - Form Template ID (Also Known As Component #) of the form to be launched when the criteria matches.
  • Selecting either the Launch from Form or Launch from PDQ check box causes additional fields to appear which allow you to select the specific form or PDQ.
  • Mode - Code for the form launching functionality.
  • Modal - When selected the form that opens retains focus, meaning that users won't be able to work in another window until the new form window closes.
  • Form Height and Form Width - Indicates the size of the window when the form launches.
  • Form Title - Allows you to control what appears in the title bar of the form window. When left blank, the component title alone is displayed.
  • Form Target - Controls if the form is opened in a new window or within the current form.
  • Form Parameters - User or system define parameters that control any number of things about the form to be launched.
  • Enabled - Indicates if this form launching action is enabled in the system or not.
