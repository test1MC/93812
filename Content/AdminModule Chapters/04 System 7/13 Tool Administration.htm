Tool Administration



Tool Administration.

 

Edit Mode: Form and Grid

Description: The Tool Administration option allows Administrators to manage the tools that appear on the Tools tab in the File Drawer. Administrators may need to use Tool Administration to provide or revoke access for a specific role. Adding and deleting tools and other details should be reserved for Libera Developers.

 

ECF Columns

  • Number - Unique ID number for the tool component. Also known as Component #. It's displayed on the Tool Administration form in the Tool # field.
  • Name - Name of the tool. It's also displayed in the Name column of the Tools tab of the File Drawer.
  • Group - How the tool is categorized within the grid and on the Tools tab of the File Drawer.
  • ACL - A comma separated list of User Roles that have the ability access and use this tool from the Tools tab of the File Drawer.
  • Module - Displays the module in which the tool is visible. For example when the value for this cell is General, the tool is available all of the time, however if it is anything else, the tool only appears when the selected module is loaded in the Data Tree.
  • Agency - No longer used.
  • Object Type - Typically related to the Module, used more with form components than with tools.
  • Flags - Component specific instructions and/or notes for the use of Libera developers. Used more with other types of components such as forms or PDQs than with tools.
  • Code and XML Code- Commands, both [Javascript] Code and XML Data Commands provide specific instructions for the component relating to many things including how/when it opens, window size, etc.
  • Stage Controls when the tool is visible to the Agency. Note: User Roles also control who can see and use the tool, so even if it is visible, it may not be available to all users.

 

ECF Grid Functionality

  • Tools are grouped by Group. You may collapse and expand groupings for faster navigation.
  • Selecting a record in the grid opens the Tool Administration form for the selected record.

Adding a New Tool

Selecting the Action Row, where it says, "Please click here to add a enw record..." opens a new Tool Administration Form.

Note: For best results, please contact Libera to request the addition of a new tool.

Deleting a Tool

Tools are deleted by Libera. Please contact Libera to delete a tool. An alternative to deleting a tool is to add !0 to the ACL (Roles) field of the Tool Administration form. All users inherently have the User Role 0. The exclamation point in front of the User Role excludes users with that role from access to the component. When the ACL is set to !0 no users, including System Administrators, have access to the component.

 

  • While you have the ability to edit data in the grid, it is easier to edit data via the Tool Administration form.
