External User Administration Form


External User Administration form.

 

Overview

Use the External User Administration form to manage external user's log on credentials as needed.

 

 

Form Functionality

  • First Name, Last Name, and Email - These fields are populated based on the user's data entry via the Portal.
  • Username - This is the name that the user uses when logging on to the system.
  • Set New Password - Use this field to reset the user's password in the event they have forgotten their current password. Optionally, you may use the Create Password button to automatically generate a new password. Setting a new password either way gives you the option to notify the user of their new password via email.
  • Password Never Expires - When selected the password doesn't expire, this is not recommended. When blank, users are prompted upon first log on (or after their password is reset) to change their password. After that they will be prompted to change their password at the interval set by your Agency.
  • Last Login Date - Displays the date the user last logged on to the system.
  • Unlock Account - If the account is locked, use this button to unlock it.
  • Custom Login Message - If the user is locked out (or terminated), this message is displayed on the login page. If the user is active, it is displayed upon log on to the system.
  • Reset User Security Questions - Select this button to reset the User Security Questions for the user. When you do this, the next time the user logs on, they are required to select and answer new security questions.
