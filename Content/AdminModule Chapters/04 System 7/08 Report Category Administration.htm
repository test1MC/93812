Report Category Administration



Report Category Administration.

 

Edit Mode: Grid

Description: The Report Category Administration option allows agency Administrators to manage the categories used to organize reports.

Adding a New Report Category

  1. Select the Action Row where it says "Please click here to add a new record..." This action opens the Description cell for editing.
  2. Enter a description for the new report category. Note: D numbers are automatically generated. There is no need to enter one.
  3. Press CTRL+ENTER or single-click into a row within the grid to add the new report category.

Your new report category now appears in the Category lookup on the Report Administration form.

Modifying a Report Category

  1. Select the report category to be modified in the grid.
  2. Press F4 or right-click to open the cell context menu. Select the Edit option from the menu. This opens the cell for editing. Alternatively, you also simply start typing in the cell. Be careful as doing so overwrites the cell contents.
  3. Press CTRL+ENTER or single-click into a row within the grid to save your changes.

Deleting a Report Category

Please contact your Supervisor or contact Libera Client Support to delete a report category.
