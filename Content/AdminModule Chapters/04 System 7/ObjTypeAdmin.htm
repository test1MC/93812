Object Type Administration.

 

Description: The Object Type Administration option allows Libera developers to manage “object types.” Object types are descriptions of various entities represented within an application such as clients, vendors, and services. This is used to associate forms and letters with the appropriate entity. The grid shows the ID, name, and a description of object types.

It is preferred that object types administration tasks be performed by application developers only, as these functions affect which forms are available in the electronic case folder.

To view an existing object type:

1. Type a key word in the filter row of the Object Type Administration ECF and press ENTER to filter the list. You can view information in the grid.

2. ID – Identification number for the object type. Do not change.

3. Name – Name of the object type. Do not change.

4. Description – Description of the object type. Do not change.
