Default PDQ Setup



Default PDQ Setup.

 

Edit Mode: Grid

Description: The Default PDQ Setup option allows Administrators to configure default PDQs for each role. The default options determine which PDQ users see when they select a module from the Application Home Page. For instance, if you want System Admin users to automatically see the View All Vendors PDQ whenever they select the Vendors option from the Application Home Page, then you would set View All Vendors as the default for their role.

The grid shows all default PDQs grouped by role. It also identifies the module and PDQ name. The application includes all default PDQs recommended by Libera. However, Administrators can change PDQs to meet their agency's needs.

Note: Users can individually change their default PDQs.

 

Adding a Default PDQ to a User Role
  1. Select the Action Row where it says, "Please click here to add a new record..." This opens the row for editing. Each cell (Role, Module, PDQ Name) has a lookup from which to select a value.


    The Action Row opened for editing with the lookup button circled and lookup displayed.
  2. Make selections for each cell, then press CTRL+ENTER or single-click into the grid to save your changes.

 

 Changing an Existing Default PDQ
  1. Open (right-click or press F4) the cell context menu in the cell that you will be changing. Select the Edit option. This opens the cell up for editing.


    Opening a cell for editing. 1) Open the cell context menu. 2) Select the Edit option. 3) The selected cell opens for editing.
  2. Make a selection from the lookup to change the contents of the cell.
  3. Press CTRL+ENTER or single-click into another row in the grid to save your changes.
