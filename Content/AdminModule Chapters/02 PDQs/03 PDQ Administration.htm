PDQ Administration.

 

Edit Mode: Form

Description: The PDQ Administration option allows Administrators to control all PDQs that are visible to users. The grid shows all PDQs grouped by module. When a specific module is expanded, you can view the PDQ ID number, a brief description of the PDQ, and all roles that can access the PDQ. Use the filter to find specific PDQs. Administrators can change the description, module, or role assignments for PDQs when appropriate. However, such changes should be made with caution, as they may affect current users of the specified PDQ.

Adding a New PDQ

Work with Libera to add new PDQs to the system.

Modifying an Existing PDQ

  1. Select the PDQ in the grid to open the PDQ Administration form.
  2. Edit the form as you would any other form. Refer to the PDQ Administration Form topic for more information on working with the PDQ Administration form.
