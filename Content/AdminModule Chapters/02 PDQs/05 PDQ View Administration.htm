PDQ View Administration.

 

Edit Mode: Grid

Description: The PDQ View Administration option allows Administrators to control the available Views options that are visible to users in the Views tab of the File Drawer. Views change the look and contents of the Electronic Case Folder (ECF). Administrators can change the description or role assignments for the Views when appropriate. However, such changes should be made with caution, as they may affect current users of the specified View. When a new role is added to the system, it should be added to all relevant Views that are not universal. New Views must be added by Libera Developers.

 

Modifying a View

  1. Open (right-click or press F4) the cell context menu in the cell that you will be changing. Select the Edit otpion. This opens the cell up for editing.
  2. Edit the cell as desired.
  3. Press CTRL+ENTER or single-click into another row in the grid to save your changes.

Note: The data in the following columns is managed by Libera. Do not change the data in these columns.
-- PDQ ID
-- View ID
-- Object Type
-- Flags
-- OnCheckViewECID
-- Stage
