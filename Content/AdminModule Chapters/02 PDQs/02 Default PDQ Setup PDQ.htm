Default PDQ Setup PDQ



Default PDQ Setup.

 

Overview

The Default PDQ Setup option allows agency application administrators to configure the default Pre-Defined Queries (PDQs) for each role. The default options determine which PDQ users see when they select a module from the Application Home Page. For instance, if you want System Admin users to automatically see the View All Vendors PDQ whenever they select the Vendors option from the Application Home Page, then you would set View All Vendors as the default for their role.

 

PDQ Results

  • The grid shows all default PDQs grouped by role. It also identifies the module and PDQ name. The application includes all default PDQs recommended by Libera. However, agency application administrators can change PDQs to meet their agency needs.
  • You can filter, group and sort the results of this grid.
  • The Default PDQ Setup PDQ loads the PDQ information into the Electronic Case Folder (ECF) when selected from the System Administration Menu menu.
  • While you can view any Favorites you may already have, you may not add new Favorites from this PDQ.
  • Favorites - Refreshes the Data Tree to show data you have marked as "favorite." To return to prior Data Tree content, select the button displaying the name of the PDQ (the first button in the list).
