Accessing System Administration

The System Administration module is available when you log on to System 7™ as a . System Administration can be accessed two ways:

  • From the Application Home Page, select the System Administration option.

    Application Home Page with System Administration selected.
  • From the PDQs tab in the File Drawer, select the System Administration Menu PDQ.

    System Administration Menu selected on the PDQs tab of the File Drawer.

The System Administration module is available when you log on to System 7™ as a . To get to it, close the Find an Account window and select the PDQs button in the Desktop Ribbon.

 


PDQs button circled in the Desktop Ribbon.

 

Filter the Name column by the word system.

Selecting the System Administration Menu option loads the System Administration module.

 


System Administration Module.

 

  • PDQs with the double pound sign ( # ) are default PDQs. You can filter by the pound sign to quickly get to the default PDQs.
