Show Flagged check box field on Case Notes

(App.CaseNote.Field.CaseNoteFlag.Visible)

 

Controls the appearance of the Flagged field on the form. When the Value field is set to True, the field appears. When it's visible on the form, users can select it to highlight the case note in the Electronic Case Folder (ECF).
