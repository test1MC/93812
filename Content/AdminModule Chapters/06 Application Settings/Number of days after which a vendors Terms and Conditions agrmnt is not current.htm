Number of days after which a vendor's Terms & Conditions agreement is no longer current

(App.Vendor.TermsAndConditions.DaysUntilExpires)

 

After the specified number of days, a Terms & Conditions event on the Vendor Information form will no longer be considered current. If a service requires Terms & Conditions then a new Terms & Conditions event will need to be created before it can be authorized. The Value represents the number of days.
