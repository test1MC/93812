Max level of account that is editable

(App.Account.MaxEditableLevel)

 

Use this setting to control the account levels that a user can edit. There are three levels in the hierarchy: 1, 2, and 3. The top level is 1. When no Value is entered, all users with access to the Account Information form can edit all levels.

If desired, use the Extended Settings of the Application Setting Form to configure this setting by user role. To do so, follow these steps:

  1. Navigate into the Extended Settings section.
  2. Go to the ACL (Roles) field, enter a user role.
  3. Enter a Value for hierarchy (1, 2, 3, or blank).
  4. Repeat for other roles as needed.
