List of attachment types that will change the indicator that pictures are provided on the Pre-Audit form

(App.Audit.ListOfPicturesProvidedTypes)

 

This application setting works with the Category lookup in the attachment section of the Pre-Audit form. The Value field accepts a comma separated list of Codes from the dictionary table LVR_Attachment_Type_CD. When a user selects one of the categories listed, the Pictures provided? field automatically displays a Y. Otherwise, it remains N.


Example of the Pictures provided? field defaulted to Y based on attachment Category selected.

PAPH is the default value. This is the code for Pre-Audit Pictures of the Home.

  • If you add category options to the list, it's helpful to have the dictionary table LVR_Attachment_Type_CD open in another window. This way you can see and copy the Codes from the table into the Value field.
