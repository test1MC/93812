ACL list of roles that can search / Add new keem jobs on the KEEM PDQs

(App.KEEM.SearchAddJobs.ACL)

The Value field accepts a comma separated list of Role IDs. Users with any one of the Role IDs listed will be able to use the Search / Add New KEEM Job button found at the bottom of the Data Tree in various PDQs in the system.

Note: These users will be able to see and use the Search / Add New KEEM Job button regardless of how the ACL list of roles that can see the buttons on the KEEM PDQs application setting is set.
