The list of roles that can add/remove Accounts

(App.Account.AddRemoveAccounts.Rolelist)

 

Allows you to control which groups of users are allowed to add or remove accounts based on User Role. The starting Value is: 6 - TechSupport, 11 - Accounting Admin, 24 - SysAdmin. You may edit the Value by removing and adding your own list or adding to the default list. The Value field accepts a comma separated list of User Roles IDs.
