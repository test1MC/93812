Show the Category lookup on attachments

(App.Attachments.Category.Visible)

 

Controls whether the Category field is displayed within the attachment section on Authorizations or not.

Note: This Application Setting is used in conjunction with the Show the 'Required Attachment for Payment' on the Service form Application Setting. Both must be set to true in order to work correctly.

  • Show the 'Required Attachment for Payment' field on the Service form
