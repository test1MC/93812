Notify staff when an account balance drops below X percent

(App.Account.NotificationPercentage)

 

This application setting works with the System Notification: Accounts within X% of Depletion. When the account drops below the specified percentage, a notification is sent to users who are subscribed to the notification. If the Value field is blank, no notifications are sent.

System Administrators manage who receives the notification via the System Notification form via the Subscribe ACL field. This field holds a comma separated list of user role IDs.
