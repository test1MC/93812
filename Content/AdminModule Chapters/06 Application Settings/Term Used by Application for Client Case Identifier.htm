The term used by the application for the Client Case Identifier

(App.Client.CaseIdentifier.Terminology)

 

Allows you to change the terminology used by your Agency for "client case identifier." The Value field displays the terminology used by your Agency. Where ever the "client case identifier" is used in the system, this term appears.

Note: This application setting can be extended so that you can use different terms for different forms or users if necessary.
