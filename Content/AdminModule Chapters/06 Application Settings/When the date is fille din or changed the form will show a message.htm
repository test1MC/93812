Causes a message to appear when the For Program field shows XX. When this setting is set to True and the For Program field shows XX* (Prog: General, Code: 0), the following message appears when users fill in the Date field and move out of it:

Please note that this scanned document form is currently set to the general program and is NOT tied to a case. Select an active program in order to assign the form to a case.

*When the For Program field appears in the form, it defaults to XX.
