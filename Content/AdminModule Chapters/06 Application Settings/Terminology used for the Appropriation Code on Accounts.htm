Terminology used for the Appropriation Code on Accounts

(App.Account.Field.AppropriationCode.Terminology)

 

Allows you to change the terminology for the Appropriation Code field on the Account Information form. This field displays an account code specific to your Agency. The default value is APPR. To change it, enter the terminology your Agency uses into the Value field.

 

  • Whether to show the APPR Field on the Account Form
