User Preferences Defined

Data Grid - Specifies How the Data Grid Behaves

Default grid zoom percentage?

Controls the size of the font in grids. When increasing the font size the grid cells will also increase in size. This means that less data fits into the visible frame and you will need to scroll more.

 

Open electronic case folder automatically?

Enables or disables opening the ECF automatically when a record is selected in the Data Tree. This increases performance when using the arrow keys to navigate up and down in the Data Tree.

When this is set to Yes, each time focus stops on a record the ECF will attempt to load, regardless of how fast you are arrowing in the grid. The affect is cumulative. The more records you arrow through, the longer it will take the ECF to load when you reach the record that you want to see it's ECF. Filtering instead of arrowing alleviates this delay.

When this is set to No, you must press ENTER or double-click the record in the Data Tree to open the ECF.

 

Prompt before add?

Enables or disables a prompt when adding a new record to a grid (provided the grid in question allows you to add new records). The prompt is a confirmation message that pops up asking you if you're sure you want to add the record. For example, "Are you sure you want to add this record?"

 

Prompt before delete?

Enables or disables a prompt when deleting a record from a grid (provided the grid in question allows you to delete records). The prompt is a confirmation message that pops up asking you if you're sure you want to delete the record. For example, "Are you sure you want to delete this record?"

 

Prompt before update?

Enables or disables a prompt when updating a record. For example, "Note: You must log out then back in before your user preference changes will take effect."

Row focus delay

Sets the delay (milliseconds) before the row focus event is fired.

 

Show filter row?

Sets the default for hiding or showing the Filter Row. You can always hide/show the Filter Row on the fly as you work in grids regardless how the default is set. See Hiding and Restoring the Filter Row.

 

Speak cell attributes?

Enables or disables speaking column/cell attributes like whether a look-up is available, editing keys, etc.

 

Speak column headers?

Enables or disables speaking column headers as you arrow through the data grid.

 

Speak format conditions?

Enables or disables speaking record and cell conditional formatting.

 

Speak grid status bar messages?

Enables or disables speaking grid status bar and progress messages such as the status of server communications, etc.

 

Speak record memo?

Enables or disables speaking additional information about tables when navigating through grids such as table headers, etc.

 

Speak table information?

Enables or disables speaking additional information about tables when navigating through grid such as table headers, etc.

 

 

Forms - Specifies How the Forms Behave

Accessibility Color Scheme

Sets the accessibility color scheme for the word processor. Forms, letters, and some reports are displayed in the word processor.

 

Accessibility Color Scheme - Visual Tooltip

Sets the accessibility color scheme for the Visual Tooltip display.


Example of Visual Tooltip with the "Normal" (white text on a green background) color scheme.

 

 

Activate arrow key navigation for 'Spreadsheet' control?

Enables or disables using the arrow keys to move through 'spreadsheet' control on forms. A spreadsheet control on a form makes data entry fields look like they are in a table. Like a table in a word processor, not like a grid used in System 7™.


Example of a spreadsheet control in a form.

 

Auto advance fields?

Enables or disables auto-advancing to next field when current field is complete. Setting it to Yes speeds up data entry. However it only works on form fields of finite size, for example a date field or a text field that accepts a specific value or a specific number of characters. When the field is completed, ie. filled up, the cursor automatically moves to the next field. Does not work with memo fields or editable regions.

Setting it Yes can cause confusion for screen reader users. Screen readers read the field label and tells them the field type. When they enter text. It reads the label, field type, and text back to them. Then automatically moves cursor focus to the next field where it starts reading all over again. It can be very confusing.

 

Auto display look-ups?

Enables or disables automatically displaying a field's look-up when you tab into an empty field.

 

Auto Save field Interval

Automatically save the current field's value (memo fields only) after X minutes (0 to not auto-save).

 

Default Font

Sets the default font for the word processor when editing memo field, editable regions, etc.

Note: The form may be configured to override this user preference.

 

Default Font Size

Sets the default font size for the word processor when editing memo fields, editable regions, etc.

Note: The form may be configured to override this user preference.

 

Default print zoom percentage?

Sets the default zoom percentage to use when printing forms (i.e. Large Print forms). You can always change the print zoom at the time you print. See Options for Printing Forms.

 

Default zoom percentage

Sets the default zoom percentage when a form loads on the screen. You can always change the zoom percentage when you are viewing the form. See Ribbon: View Tab.

 

Field Tooltip Display Mode

Sets what is displayed in form field tooltips as well as how the record tooltip is included.

  • Only field Tooltip - Regardless of the section the field is in, the tooltip displays only the tooltip. For example, "Title."
  • Record Tooltip then Field Tooltip - In a multiple-section section, the tooltip would display the record title with a number followed by the tooltip text. For example, "Client Name History 1 - Title."
  • Field Tooltip then Record Tooltip - In a multiple-section section, the tooltip would display the field tooltip followed by the record title and number. For example, "Title - Client Name History 1."

 

Goto Field Sort Order

Sets the order of the fields in the Goto Field List. You can select between Alphabetical or Form Order. See Goto Field List.

 

Highlight current section?

Enables or disables highlighting the current section, if the section allows adding records. When set to Yes, when cursor forcus moves into a multiple-section section, the background color of the entire section changes to a pale yellow.

 

Highlight field text?

Enables or disables highlighting the contents of a field when selected (has cursor focus). When set to Yes, when the cursor moves into the field all of the text in the field is highlighted blue. When set to No, the cursor is placed at the beginning of the field.

Note: When all of the text is highlighted, you can easily overwrite it. If you didn't intend to overwrite all of the text press CTRL+Z before cursor focus leaves the field. This will "undo" the change. You can only "undo" the change as long as the cursor is still in the field. Once the cursor moves out of the field you will no longer be able to "undo" any changes.

 

Prompt before adding section?

Enables or disables a prompt when adding a multiple-section section. For example, "Do you want to add a new [Section Name] section?"

 

Prompt before deleting section?

Enables or disables a prompt when adding a multiple-section section. For example, "Are you sure you want to delete [Section Name #]? Warning: This action cannot be undone."

 

Refresh grid after adding a new form record?

Enables or disables automatically refreshing a grid to show the newly added form record. When set to Yes, grids automatically refresh when a new record is added. Otherwise they don't.

 

Skip reference fields?

Enables or disable skipping reference fields when tabbing. A reference field is a read-only field. When set to Yes, keyboard navigation is faster because the cursor will not stop on read only fields.

 

Speak field attributes?

Enables or disables speaking field attributes like whether a picklist/look-up is available, etc.

 

Speak field description?

Enables or disables speaking a field's long description rather than its title/tooltip. The field title is still displayed in the Go To Field List though.

 

Speak field names?

Enables or disables speaking field names as you move between fields on a form.

 

Speak form narrative?

Enables or disables speaking form narrative when a form opens. This is all of the background stuff that sighted users do not see. It's a lot of information which isn't necessarily helpful.

 

Speak form section messages?

Enables or disables speaking additional information related to a section on a form.

 

View document attachment previews?

Enables or disables viewing document attachment previews within a form. When set to yes, a preview is shown otherwise the attached file is displayed as a link. There may be a slight performance increase if set to No.

 

Visual Tooltip Display Duration

Sets how long the visual tooltip is displayed. This can be used to provide a field label within a magnified view (with screen magnifier software). To never display the tooltip, enter 0 (zero). To display forever, enter -1. If set to never display, you can always turn have the tooltip displayed on a case-by-case basis by pressing CTRL+1 when you're working in a form. The hotkey CTRL+1 toggles the tooltip on and off.

 

 

General - General User Preferences

Auto-launch dashboard at login?

Enables or disables auto-launching of the Dashboard at login (if available).

Note: The Dashboard may be configured by your System Administrator to always display.

 

Auto-resize Desktop Window?

Enables or disables auto-resizing the System 7 Desktop window to fit the screen. When set to Yes, when you log on the desktop will always open maximized in your primary monitor (if you are working with multiple monitors).

 

Focus onto Data Tree 'Type To Find'?

When set to Yes, cursor focus will be in the Data Tree Search Pane field Type to Find, when available, by default. If not available it will be in the first cell of the first column of the Data Tree. When set to No, cursor focus is in the first cell of the first column of the Data Tree by default.

 

Preview Pane enabled?

Enables or disables the Preview Pane (form area) in the Desktop.

 

Prompt before closing System 7 Desktop?

Enables or disables displaying a prompt before you can close the last open System 7 Desktop window. It's not unusual to have multiple System 7 windows open at one time. Without the prompt, it's easy to accidentally close out of all System 7 windows even if that's not what you wanted.

 

Screen Reader Software

Specifies which screen reading software use.

 

Show Data Tree Search Pane?

Enables or disables the Data Tree Search Pane. See Data Tree Search Pane.

 

Speak user interface messages?

Speaks user interface messages such as frame information, hotkeys, etc.

 

Time Zone

Sets your time zone in the system.
