User Favorites

Overview

System 7™ allows you to mark Data Tree records as "favorites". Similar to book marking. When you mark a record as a favorite, you can access it again from any PDQ that has a Favorites button in the Data Tree. You can add or remove records from your favorites at any time.

 

Getting to the Favorites Button


Favorites button circled in the Data Tree.

 

The Favorites button is located at the bottom of the Data Tree. Please note that it is not available in every PDQ.

  1. With cursor focus in the grid, press CTRL+SHIFT+A to activate this button immediately, or single-click it with the mouse. The Data Tree refreshes to display records you have marked as a favorite.
  2. To return to original grid, press CTRL+SHIFT+B, then ARROW to the title of the original PDQ and press ENTER. Alternatively, single-click the button displaying the title of the original PDQ.


Example of the title of the original PDQ button circled at the bottom of the Favorites Data Tree.

 

Creating Favorites

  1. Right-click on any record in the Data Tree to open the cell context menu. Select the Add Favorite... option or, with cursor focus on the record, press CTRL+D.
  2. The Add Item To Your Favorites List dialog window opens. By default, cursor focus is in the Description field. You may edit the description if desired. The contents of this field is what appears in the Data Tree of your Favorites list.
  3. Use the Add Favorite button to add the selected record to your Favorites list.
  4. If you change your mind, use the Cancel button to not add a new record to your Favorites list.

Note: Even if the Favorites button is visible in the Data Tree, not every PDQ allows you to add records to your Favorites list. When this is true, the following message appears when you attempt to add a record: A user favorite cannot be added from this table.

 

Deleting Favorites

  1. While viewing your Favorites, select the record in the Data Tree that you want to remove.
  2. Right-click or press F4 to open the cell context menu. Select the Delete option.
