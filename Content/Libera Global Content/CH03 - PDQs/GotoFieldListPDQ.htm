Goto Field List



Example of a Go To Field List.

 

Overview

The Go To Field List PDQ is a dynamic PDQ that lists all the visible fields on a form at the time it's opened. It allows users to jump to a field anywhere on the form without having to tab through form to reach the field. It also displays the value for the field and the field order as viewed on the form. The Go To Field List is accessed from the Navigation Grouping of the Home Ribbon.

PDQ Results

  • Forms that work with many tables in the database take more time to open the Go To Field List.
  • When you access the Go To Field List before navigating into the form cursor focus is in the Filter Row in the grid.
  • To jump to a field on the form, select the field in the grid then double-click or press ENTER.
  • The Value column displays the value in the field, if any. If the cell in the grid of the Value column is blank, then the field is blank on the form.
