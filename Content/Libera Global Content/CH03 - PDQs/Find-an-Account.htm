Find an Account


Find an Account window.

 

Overview

The Find an Account window is a fast way to find a specific account. The window is divided into two sections: Search and Recent Accounts.

 

Search Functionality


Search example for the name "Smith" with results.

 

  • Search for - Use this lookup to change your search target. The default search target is "All Accounts." The lookup displays many different kinds of account types from which to select. To select an account from the lookup, double-click it. This closes the lookup and shows your selection in the Search for field.
  • Account No, Owner Name, SSN, Etc... - This is the search field. Enter the account number, account owner's name or SSN, etc. then select the search button (or press ENTER) to start the search.
  • Select the account record in the grid to see its details. This action closes the Find An Account window and opens the Account Application if the account is in Incomplete or Pending Approval status, or the Account Menu for any other status.

 

Recent Accounts Functionality

  • The Recent Accounts side of the Find an Account window displays accounts with which you have recently worked. The account you most recently worked with is listed first in the grid.

 

  • You can re-size the Find an Account window by dragging the window borders. This allows you to see more grid columns in both sides of the window.
  • Make use for the vertical and horizontal scroll bars for both grids in the window.
