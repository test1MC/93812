System Administration Menu



An example of the System Administration Menu within the System.

 

Overview

Use the System Administration Menu PDQ to manage and configure your System 7™ application. This PDQ loads when you select the System Administration option from the Application Home Page or the System Administration Menu PDQ from the PDQs tab of the File Drawer.

There are many things you can do via this PDQ however, some of the most common tasks are:

  • Employee Administration - add new employees, change their contact information, add termination dates, manage their user roles, etc.
  • Dictionary Administration - edit the content of lookups in the system.
  • PDQ, Form, Report Administration - manage access to PDQs, forms, and reports within the system.

 

PDQ Results

  • The records in the Data Tree are grouped by categories of functionality. Think of each record as an "option" that you can select to manage some aspect of the system.
  • Every option in the Data Tree loads its own PDQ in the ECF. The functionality of the grid displayed in the ECF varies by the option selected.
  • Additional PDQ functionality such as grouping and sorting is not available for this PDQ.
