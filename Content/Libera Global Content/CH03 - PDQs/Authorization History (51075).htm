Authorization History


Authorization History PDQ.

Overview

The Authorization History (PDQ) is useful for quickly tracking fiscal data related to authorizations.

When accessed via an Electronic Case Folder (ECF) View, it displays transaction history for all Authorizations on file in a 's ECF. When accessed via the Tools menu of an Authorization form, it displays the transaction history for only that authorization.

When accessed via the Tools menu of an Authorization form, it displays the transaction history for only that authorization.

If an authorization has at least one payment associated with it, its record can be expanded in order to view details of each payment. These details include the name and number of the account charged, the date the payment was batched, and the type of payment.

Expand a record by either selecting it and pressing SPACEBAR or single-clicking the plus sign to the left of the record. Authorization and forms can be opened by double-clicking or pressing ENTER on the record in the grid.

 

 

Viewing the History of a Specific Authorization

  1. Open the Authorization.
  2. Select Tools from the menu at the top of the form.
  3. Select the Authorization History option.

A new window opens displaying the Authorization History PDQ.

 

Additional Information

  • Negative payments (credits) and liquidations (decreases) are displayed in parentheses.
  • Data is grouped by service line item. All grouped items are expanded by default. Use the SPACEBAR or single-click the plus/minus sign to collapse or expand the grouped rows.
  • Suffix - Shows the code related to the Type of the transaction. The code is alphanumeric. The letter indicates the type of transaction. The number indicates the order of the transaction.
