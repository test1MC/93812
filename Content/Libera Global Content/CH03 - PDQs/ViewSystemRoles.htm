View System Roles



View System Roles PDQ.

 

Overview

The View System Roles PDQ is a System Administrator tool used to find out how the selected User Role is applied to components. Data is returned grouped by component category: Forms, PDQs, and Reports. Expand each category grouping to see which components the User Role can access.

 

PDQ Parameters

  • You must select a User Role from the Role lookup in the parameter window in order to run this PDQ. The User Role you select appears in the grid title bar of the Data Tree.



Example of the selected User Role showing in the grid title bar of the Data Tree..

 

PDQ Results

  • The Data Tree displays components grouped by category: Forms, PDQs, Reports. You may not ungroup.
