User Preferences

Accessing Your User Preferences

To access your User Preferences select the Preferences button in the Profile Grouping of the Desktop Ribbon.


Preferences button circled in the Profile grouping of the Desktop Ribbon.

 

Overview

User Preferences allow you to change how the system looks and works for you. For example, you can change the size of the font in grids and forms, the color in forms, and accessibility functionality.

 

Functionality

  • Find the preference you would like to change by filtering or scrolling through the options.
  • Select the Value cell of the option you would like to change.
  • To edit, start typing or press F2 in the Value cell to open it for edting. Alternatively, press ALT+F1 to open the lookup. The lookup presents a list of available options specific to the selected option.
  • Once you have made the change, click off the option or press the UP ARROW. A confirmation message appears, "Do you want to save your changes?" Select the OK button or press the SPACEBAR to activate the OK button.
  • Repeat until all desired changes have been made.
  • You must log out and log in to the system again to see your changes.
