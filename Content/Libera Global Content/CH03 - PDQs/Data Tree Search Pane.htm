Data Tree Search Pane


Data Tree Search Pane.

 

The Data Tree Search Pane is an alternative way to search the grid for data. The best time to use it is when the contents of the Data Tree are grouped into parent and child grids. Using the Data Tree Search Pane allows you to search/filter the child grids. The Data Tree Search Pane consists of a text box to enter search criteria, search status, and the Show Search Options button. You may TAB between the three or point and click. Access the Data Tree Search Pane by pressing ALT+T when cursor focus is in the grid of the Data Tree or click into the text box.

The default search for data is Type [Column Header of First Column in Parent Table] that is like. You can change the search target by selecting the Show Search Options button and changing the Field or the Operation. Opening the Field drop down shows all column headers for the parent and child tables.

Show Search Options is a button that opens to show you the search options available to you. You may TAB to the button and then press ENTER, press ALT+S, or click the black arrow on the yellow tab pointing to the left. Note: ALT+S toggles the option tab open and closed.

 

Data Tree Search Pane Not Visible?

If the Data Tree Search Pane is not visible to you, it may have been disabled in your User Preferences. To check, follow the steps below.

  1. From anywhere in the Desktop, press CTRL+SHIFT+P (P for Preferences) or, single-click the User Preferences button in the Navigation Bar. Tip - when you hover over the buttons in the Navigation Bar, the button name and keyboard shortcut is displayed. This action opens the User Preferences window.
  2. The User Preferences window displays user preference options in a grid consisting of two columns and multiple rows grouped by areas of functionality. Filter the Name column by the word search.
  3. This returns one option, Show Data Tree Search Pane?. Look in the Value column, if set to No, then the Data Tree Search Pane is hidden from the Desktop. To change it, click into or with cursor focus in the cell, type the word Yes.

Note: For the setting change to take effect, you must log out of the system and back in again.

 

 Using the Data Tree Search Pane

Type a search value in the text box and then press ENTER. The default search for data is Type [Column Header of First Column in Parent Table] that is like. Pressing ENTER activates the search which filters the Data Tree to show results matching the search criteria entered.

When the search results are available cursor focus moves to the Data Tree. From there you can navigate via ARROW keys and access the Electronic Case Folder (ECF). Click into the ECF or press CTRL+SHIFT+T.

To return to the Search Box, click into the text box or press ALT+T when cursor focus is in the Data Tree.

Note: CTRL+SHIFT+T toggles back and forth between the Data Tree and the ECF.

 

 Changing the Search Options

To change the search options click the Show Search Options arrow or press ALT+S when cursor focus is in the Search Box. This opens the Search Tab. You may change the field (think column header) searched and the search operation.

  1. When the Search Options tab is open the Field field is selected by default. Single-click the down arrow or press ALT+DOWN ARROW to see the list of available options. Use the mouse or the ARROW keys to scroll through the options. Single-click or press ENTER on an option to select.
  2. When the Search Options tab is open select the Operation field or press the TAB key or ALT+O to change focus to the Operation field. Then single-click the down arrow or press ALT+DOWN arrow to open the option list. Single-click or press ENTER on an option to select. Operations determine how the entered search criteria affects the search results.

 

 Operations Defined
  • Between - When using this operation enter the search criteria as follows. Type two search values with the word and between them. The search results will include the two values and everything between.
  • Equal To - Search results will be equal to the search criteria entered.
  • Greater Than - Search results will be greater than the search criteria entered.
  • Greater Than or Equal To - Search results will be greater than or equal to the search criteria entered.
  • Is Empty - Search results will show records with data missing in the column searched.
  • Is Null - Search results will show records with data missing in the column searched.
  • Less Than - Search results will be less than the search criteria entered.
  • Less Than or Equal To - Search results will be less than or equal to the search criteria entered
  • Like - This is the default operation. Search results will contain the search criteria anywhere. For example at the beginning, middle, or end.
  • Not Between - Similar to Between but the results are reversed. The search results will exclude the search values entered and everything between the two values.
  • Not Equal To - Search results will exclude the search criteria entered.
  • Not Is Empty - Search results will exclude records with data missing in the column searched.
  • Is Not Null - Search results will exclude records with data missing in the column searched.
  • Not Like - Search results will exclude records that contain the search criteria entered.
