Select Dashboard Items


Select Dashboard Items PDQ.


Overview

The Select Dashboard Items PDQ shows you all available Items to display on the dashboard. This grid is accessed via the dashboard and via the Dashboard Configuration section of the Employee Information form.

 

PDQ Results

  • Select one or more records in the grid by selecting the check box of the option(s) you want displayed in your dashboard. Please refer to the Multiple Select Lookups and Grids topic for more information.
  • When working in the grid via the Select Items button of the dashboard, select options by select options by single-clicking the check box or press CTRL+SPACEBAR. When you close the window the selection options appear in the Dashboard.
  • When working in the grid via the Employee Information form, select options by single-clicking the check box or navigate to the check box column then press the SPACEBAR key. Then next time the employee refreshes their dashboard the new selections will appear.
  • Remove selection options by clearing the check box. Single-click with the mouse or press the SPACEBAR or CTRL+SPACEBAR to clear check boxes.
