Navigation Menu Bar


Desktop with Navigation Menu Bar circled.

The Navigation Menu Bar is located in the upper left hand corner of the system. The Navigation Menu Bar is always visible in the Desktop. It consist of the following buttons:

Navigation Menu Bar Buttons Table
Button Name Hotkey Description
Application Home CTRL+SHIFT+HOME Creates a new window/tab displaying the Application Home Page.
Open PDQs CTRL+SHIFT+Q Launches the File Drawer with the PDQs TAB selected.
Create New Form CTRL+SHIFT+N Launches the File Drawer with the Forms TAB selected.
Keyboard Shortcut CTRL+SHIFT+K Launches a new window with all of the System 7™ keyboard shortcuts listed and explained.
User Preferences CTRL+SHIFT+P Launches a new window with all of the available, customizable user preferences.
System 7 Program Flow   Launches a new window with available Program Form Flows.
System 7 Help CTRL+SHIFT+H Launches a new window displaying theSystem 7™ User Guide.
