System 7™ User Guide

The System 7 User Guide is designed to give you an overview of how the system is designed to work. It provides an in depth look into the basic fundamentals of the system for a first time user including such topics as how to navigate the system, grids and their functionality, an explanation of all the different parts of the system, and much more.

The System 7 User Guide should be reviewed and used along with this training manual to gain a complete understanding of how the system is designed to work.

How to access the System 7 User Guide

  1. Access the File Drawer (the yellow tab located on the far right hand side of the screen). Alternatively, you can press CTRL+SHIFT+O.

    Open File Drawer button circled in the Desktop.
  2. Type the word “Help” in the Filter Row (the field(s) located directly beneath the column headers) of the Name column and press ENTER. This is known as filtering – the process of narrowing down search results. Filtering is your friend! Select the View All Help Documents option from the results. Alternatively, you can use the ARROW keys to navigate to the View All Help Documents PDQ option and press ENTER.

    PDQs tab filtered by the word "help" with View All Help Documents selected.
  3. The Help System Desktop window opens. The Data Tree displays various System 7™ user guides. Select the System 7 User Guide option. Alternatively, you can use the ARROW keys to navigate to the System 7 User Guide option and press the SPACEBAR to expand the grouping and see the Table of Contents for the selected user guide. Press the + button, or SPACEBAR on each sub-topic to expand the grouping. Select a topic title to display the content in the Preview Pane.

    Help Desktop with steps to open content outlined.
    1. Expand the Common: System 7 grouped row.
    2. Expand the Document: System 7(TM) User Guide row.
    3. Select a topic in the Data Tree to load the Preview Pane.
