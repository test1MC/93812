File Drawer


Desktop with File Drawer open.

The File Drawer is located on the far right side of the screen and when expanded, covers the Electronic Case Folder (ECF). The File Drawer is made up of 8 TABs: PDQs, My PDQs, Views, Forms, Letters, Reports, Tools and Links. Additional information on the File Drawer and the 8 TABs can be found in the online System 7™ User Guide under System 7™ Navigation > File Drawer.
