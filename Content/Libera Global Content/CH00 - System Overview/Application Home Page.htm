Application Home Page


Screenshot of the Application Home Page.

The Application Home Page is the first screen that you encounter upon logging into the system. Depending on system access and system setup, the Application Home Page displays different content for different users.

Some users will see more options than others and some users will see less. If you don’t have access to an option that you believe you should have access to, please contact your Supervisor or System Administrator for assistance.

Each of the options on the Application Home Page has a specific Pre-Defined Query (PDQ) linked to it, so when the option is selected, the system launches the associated Default PDQ.
