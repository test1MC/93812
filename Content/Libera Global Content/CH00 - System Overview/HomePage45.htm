Home Page


Start Page with the Find An Account window displayed.

 

The Home Page is the first screen that you encounter upon logging into the system. The system displays the Find an Account window. You can immediately search for the account with which to work. If you have another task to do, you can close the Find an Account window and select another option from the Ribbon: Desktop Tab.

 

You can get back to the Home Page at any time by selecting the Home button in the Desktop Ribbon.
