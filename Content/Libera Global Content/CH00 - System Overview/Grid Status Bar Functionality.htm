Grid Status Bar Functionality

The Grid Status Bar is located along the bottom of all grids. Each grid in the system has its own status bar which consists of several buttons:

 Grid Status Bar Buttons Table
Button Name Hotkey Description
Refresh CTRL+R Refreshes the information in the grid, versus refreshing the entire screen/form/etc.
Load More Data CTRL+L Loads the next group of records, up to 250 records at a time, when there are more than 250 records in the grid.
Export Data CTRL+E This button provides you with options to export the grid data. You will learn more about exporting grids in the online System 7™ User Guide under Grids – Behold the Power > Printing Grid Data.
Show Help About This PDQ CTRL+H This button launches a new window with help documentation specific to the PDQ/grid being viewed.
General PDQ Grid Help CTRL+ALT+H This button launches a new window with help documentation specific to grid functionality and navigation.
