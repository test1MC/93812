Troubleshooting

Help, I can't find the File Drawer!

From the Desktop, there are three ways to open the File Drawer:

  1. Select the Open File Drawer button. Look for the yellow tab on the right edge of the window near the top of the screen. Select the black arrow pointing to the left.

    Open File Drawer button circled in File Drawer.
  2. Press CTRL+SHIFT+O.
  3. Select the Open PDQs or Create New Form button in the Navigation Menu Bar. These buttons open the File Drawer to the PDQs or Forms tab respectively. You can navigate to other tabs from there.

    File Drawer and Create New Form buttons circled in the Navigation Menu Bar.
 Help, when I print a grid I get a screen shot instead!

To print the contents of a grid, use the Export Data button located in the status bar of the grid. Select the button with your mouse or with cursor focus in the grid, press CTRL+E.

  1. Select the Export Data button or press CTRL+E. This opens the export menu.
  2. In the menu, select the Print Preview option, then select Export Now. This opens a new window, showing you a print preview of the grid contents.
  3. In the Print Preview window, select the File menu, then select the Print option or press CTRL+P to to open the Print Dialog window. Follow the the steps on the screen to print.
