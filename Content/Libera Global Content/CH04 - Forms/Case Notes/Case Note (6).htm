Overview

The and forms allow you to enter information about contact with the , vendor, employer, etc. or notes about the case. forms are found at the top/bottom of many forms.

Form Functionality

  • Name – Automatically populates with the selected name.
  • The Date field automatically populates with today's date however, you can change it within Agency set parameters if necessary.

  • Description – Select or type your own brief description of the content of the note. The contents of the Description field appear in the Description column of the Electronic Case Folder (ECF). Depending on your Agency's settings you may be able to select multiple descriptions, if so an additional Desription field appears below the first Description field.

  • Status - Initially blank and read-only. It becomes editable when you are doing a manual status change. When doing a status change you must select a "status change" option from the Description lookup, this opens the Status field for editing. Status changes only apply to .
  • Note - You can enter any amount of text into this field. It's a memo field which allows you to format the text if desired. You can paste information from other sources as well.
  • If you're doing a status change, filter the Description column in the Description lookup by the words status change.
  • If you've completed the form and need to add another note for the same , open the File menu and select the New Note... option. This opens a new blank and closes the first one.
