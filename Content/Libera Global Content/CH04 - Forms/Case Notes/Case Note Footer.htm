Unless otherwise specified, using the section is optional. A blank case note section is viewable when the form opens. When filled in, the case note is always accessible at the bottom of the form and in the Electronic Case Folder (ECF). Select the Move To Previous Note button at the bottom of the section. You can add multiple new case notes via the Case Note Footer. Once the blank note is filled in the New Note button appears at the bottom of the section. Select the New Note button at the bottom of the section.

Note: The terminology for this section is set by your Agency. The Case Note Footer is known by many names: Contact Note Footer, Comment Footer, etc.
