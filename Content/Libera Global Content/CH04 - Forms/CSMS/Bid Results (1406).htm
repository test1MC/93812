Top portion of the KEEM - Bid Results form.

Overview

  • A form collects quotes from up to three (3) for a specific Job.
  • Multiple forms can be created if more than three (3) quotes are needed. The Add More Bids button appears when three with quotes have been added to the form. When the button is selected, a new form opens.
  • When the form is approved an Authorization form opens for the selected .

  • If you have rights to see the Vendor Information form for the , the Vendor Info button appears at the bottom of the column when a is added to the form either via the field or when you manually add a new to the form. This button opens the Vendor Information form.

  • Change Winning Bid Amount - This button appears after the form has been approved. Selecting the button opens the Bid History PDQ which displays the history of the winning bid.

System Administration

The ACLs for the form should be set to make sure designated users can access the form.

  1. Go to the System Administration module.
  2. Open the Form Administration table.
  3. Filter the Template ID column by the form component # 1406.
  4. Select the form in the table. This opens the Form Administration form. Navigate to the Security Access section of the form.
  5. Adjust the ACL fields as desired for your Agency.
