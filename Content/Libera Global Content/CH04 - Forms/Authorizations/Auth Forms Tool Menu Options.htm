Authorization Forms Tool Menu Options

Review

Opens the form. Designated employees use this form to pay the Authorization.

 

Authorization History

Opens the Authorization History PDQ. This PDQ displays transactions by line item with subtotals. It also displays account information and other relevant information.


Authorization History PDQ.

 

Running Totals

Opens the Running Totals PDQ. This PDQ displays all transactions by line item in chronological order (oldest to newest) with a running total.


Running Totals PDQ.

 

Approval History

Opens the Authorization Approval History PDQ. Each time a signature line is signed it creates a record in this PDQ. The signature records are displayed in chronological order (oldest to newest) in the grid.


Approval History PDQ.

 

Apply Penalty

Opens the Contractor Penalty form. This form displays a section for each service line item on the Authorization. If you enter a Penalty Amount you must also enter a Penalty Reason. The penalty is applied when you select the Save button. The form only displays the total penalty amount. To see the individual records, open the Penalty History*. This option only appears on forms.

 

Penalty History

Opens the Penalty History PDQ. This PDQ displays the individual penalty records for each service line item in the Authorization. This option only appears on forms.


Penalty History PDQ.

 

Cancel Balance

This option allows designated users* to cancel the outstanding balance on the Authorization. When selected the following message appears: Are you sure you wish to Cancel the Remaining Balance and Lock the Authorization? Selecting the OK button cancels the remaining balance of the authorization.

Upon selecting the OK button the following message appears: The Remaining Balance has been canceled. Please note that the form will still appear to display the original amounts. See Authorization History for details.

Void ___________ Authorization

This option allows designated users* to void the Authorization form. When you select this option the following message appears: Are you sure you with to reduce the total to zero and delete the [authorization form name] from the ECF? Selecting the OK button zero's out the amount(s) and removes the form from the ECF.

 

Note: *If you aren't a designated user of a described option, you won't see that option on the Tools menu.
