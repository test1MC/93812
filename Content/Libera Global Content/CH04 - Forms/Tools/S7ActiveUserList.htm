System 7 Active User List

Accessing Tools

 


System 7 Active User List.

 

The System 7 Active User List is used by System Administrators to see who is logged into the system. It opens in the System 7 Administration Console.

 

Columns Defined

  • ID - Displays the users ID from the Employee Administration table.
  • User Name - Name of employee, first and last, logged on followed by their User Name from their Employee Information form.
  • First Request Time - Date/Time stamp when they logged on.
  • Last Request Time - Date/Time stamp of last activity in the system.
  • Last URL - Last location in the system. This data may be scrambled for security purposes.
  • Session ID - System generated unique identification for the session.
  • IP Address - IP address of the user's computer.
