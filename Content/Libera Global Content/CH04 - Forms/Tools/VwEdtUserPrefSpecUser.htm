View/Edit User Preferences for a Specific User

Accessing Tools

 

Overview


View/Edit User Preferences for a Specific User tool.

 

This View/Edit User Preferences for a Specific User tool displays a specific user's preferences and allows you to view and edit the preferences. This tool is helpful when setting up a user in System 7™ for the first time, especially accessibility users where the preferences need to be set for System 7™ for properly function.

 

Parameter Window


User Preferences for a Specific User Parameter window.

 

The User ID lookup provides you with a list of users in System 7™ whose preferences you wish to review. You must use the lookup to select the user. When you return to the parameter window the User ID will be filled in, this is NOT the Worker ID, it is the unique number assigned to the user in System 7™.

 

Functionality

This tool functions the same way as setting your own User Preferences does. The only difference is that you're setting preferences for a user other than yourself.
