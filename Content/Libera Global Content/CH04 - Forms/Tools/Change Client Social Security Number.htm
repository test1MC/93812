Change Client Social Security Number


Change Client Social Security Number window.

 

Overview

Use the Change Client Social Security Number tool to change the 's Social Security Number. Your System Administrator controls access to the Change Client Social Security Number tool. Only designated users have the ability to use it.

 

Where to Find the Change Client Social Security Number Tool

The Change Client Social Security Number tool appears as an option on the Tools tab in the File Drawer when you have a selected in the Data Tree.

The Change Client Social Security Number tool appears as an option on the Tools tab in the File Drawer when you have a Job selected in the Data Tree.

 

 

How to open the Change Client Social Security Number Tool

  1. Select a in the Data Tree.
  2. Open the File Drawer to the Tools tab.
  3. Filter the Tool Name column by the word social. Select the Change Client Social Security Number option.

 

Form Fields

  • Client Social Security Number - Automatically populates with the selected 's current SSN.
  • Client Name - Automatically populates with the selected 's name.
  • New Social Security Number - Enter the new Social Security Number.
  • Change Client Social Security Number - When this button is selected the following message appears:

    Change Client Social Security Number confirmation.
    Select the OK button to close the message.
  • Cancel - Selecting this button cancels the SSN change.
