Manage Your Notification Subscriptions

To access the Manage Your Notification Subscriptions tool, select the Tools button in the Account Management grouping of the Desktop Ribbon. Then filter the Tools window by the word manage.

 


Accessing the Manage Your Notification Subscriptions tool.

 

Selecting Manage Your Notification Subscriptions opens the tool in a new window.

 

Overview

The Manage Your Notification Subscriptions tool allows you to sign up for and cancel notification subscriptions. A notification is a message you receive via e-mail regarding an action within the system.

 


Manage Your Notification Subscriptions tool.

 

The Available System Notifications vary depending on which system notifications used by your agency. This is controlled within the System Notifications option in the System Administration module by your System Administrator. Only active notifications appear in the tool.

You can sign up to receive any of the available notifications by responding Y (Yes) to the Subscribe? field for notification. This opens the Sent To field for editing. Leave it blank to use the e-mail address the system has on file for you. To stop sending a notification, change the response to N (No) and clear the Send To field.

Notifications send a system generated e-mail to you any time the criteria of the notification is met.
