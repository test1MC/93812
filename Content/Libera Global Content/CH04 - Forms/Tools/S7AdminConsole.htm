System 7 Administration Console

Accessing Tools

 

Note: The System Administration Console is accessible via the System 7 Log Viewer or System 7 Active User List tool.

 

Database Console


System 7 Database Console.

 

Knowledge of SQL, system tables, and column names is required to successfully use the Database Console. Designated users, such as System Administrators, use it to inves tigate data issues. It allows you to verify data at the database table level but not change any data. There may be times when Libera will provide information for you to use with the Database Console.

  • Database Profile - A connection to a specific database and schema combination. The available profiles vary by agency. Several of the common profiles are described below:
    • sys7db - This is the default profile. It's the main database for storing system data.
    • sys7dict-direct - Dictionary database
    • sys7rcc-data - Report Control Center table administration.
    • sys7db-archive - Stores archived data.
  • Command Type
    • Select - Used anytime you want to run a SQL query.
    • DataCommands - Allow you to run a script provided by Libera. It's in Libera's LForm proprietary language.
  • Output Format
    • Interactive Grid - Results are displayed in a grid.
    • Excel CSV, HTML, XML - Opens the results in a file in the specified format.
  • Timeout - The number of seconds after which the command will automatically fail if it hasn't completed within the specified timeframe.
  • Enter your Commands - This is where you enter the SQL query or the data command script.
  • Submit - Runs the instructions in the Enter your commands field.
  • Results - Where the grid is displayed or if you're running data commands it displays the results of the script and any related messages.

 

 Edit Check Console


System 7 Edit Check Console.

 

Allows you to run a function within the system. Edit checks can do anything from displaying information to making changes in the system. However, you have to have an existing Edit Check ID.

 

 Memory Utilization Report


System 7 Memory Utilization Report.

 

The Memory Utilization Report is an advanced administration tool to be used at the direction of Libera. It provides visibility into memory utilization of the web and application tiers. Shows information on caching data and system components.

 

 Network Speed Test


System 7 Network Speed Test.

 

The Network Speed Test measures the network connection speed from your computer to the System 7 servers.

  • HTTP Latency - The time it takes for a single request to be sent to System 7 and back again regardless of how much data is sent/received. The longer the time, the slower all pages and images will load in System 7.
  • Download Speed - How fast data can be downloaded from System 7 to your browser. The speed is reported in MB/s (megabytes per second) or KB/s (kilobytes persecond). The higher the number, the better.
  • Upload Speed - How fast data can be uploaded (sent) from your PC/browser to the System 7 servers. The speed is reported in MB/s (megabytes per second) or KB/s (kilobytes per second). The higher the number, the better. If it says "Very Fast," it means that it was too fast to measure.
