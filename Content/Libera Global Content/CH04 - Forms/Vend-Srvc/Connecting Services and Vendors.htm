Connecting Services and Vendors

Vendors and Services are added to the system by designated staff. This happens via the View All Vendors and View All Services PDQs.

Both the Service Information and the Vendor Information forms must be approved. Then services must be linked to vendors. This is done in the vendor ECF with the Vendor's Services form. This form must be activated and approved.

All of these forms have many configuration options that allow your Agency to control when vendor service combinations are used in the system. Once the vendor service combinations are approved they can be added to plans and authorizations by users.

Note: You have to have access to the View All Vendors and View All Services PDQs to work with vendors and services.

Checking the Vendor and Service Setup

Is the Vendor an active and approved vendor?
  1. Open the File Drawer to the PDQs tab. Filter for the View All Vendors PDQ. Run the PDQ.
  2. Select the vendor in question.
  3. Open the Vendor Information form in the Electronic Case Folder (ECF).
  4. If the vendor is not active and approved there will be instructions on how to activate and approve the vendor in the Vendor Events section.

 

 Is there an active Service?
  1. Open the File Drawer to the PDQs tab. Filter for the View All Services PDQ. Run the PDQ.
  2. Open the service in question.
  3. Open the Service Information form in the Electronic Case Folder (ECF).
  4. If the service is not active there will be instructions on how to activate the service in the Service Events section.

 

 Is there an active Vendor's Services form in the Vendor's Electronic Case Folder (ECF)?
  1. Open the File Drawer to the PDQs tab. Filter for the View All Vendors PDQ. Run the PDQ.
  2. Select the vendor in question.
  3. Review the vendor's Electronic Case Folder (ECF).
  4. If you do not see a Vendor's Services form in the vendor's ECF for the service you need, add it via the Action Row where it says "Please click here to add a new form."
  5. If there is a Vendor's Services form for the service you need in the vendor's ECF open it.
  6. If the Vendor's Services is approved, the Summary tab of the Vendor Service Events section displays green text telling you the form is approved. If it's not approved, the Summary tab of the Vendor Service Events section displays red text telling you how to activate and approve the form.
