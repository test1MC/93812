Using the Context Menu to Group Data

You may find this method a little easier to use than grouping grid data by dragging and dropping columns.

Note: This method requires you to follow these steps on the column you wish to group data by.

  1. Right-click the column header row to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select the Group By This Field option.



Column header context menu with the Group By This Field option circled.

Ungrouping Data

  1. Right-click the column header row to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select the Group By This Field option.



Column header context menu with the Ungroup This Field option circled.

 

  • You can also right-click on the City column header in the Group By Box to open the column header context menu.
  • You can refresh the entire application window by pressing F5. This resets the window to its original state. Note: It will undo ALL changes you may have made.
