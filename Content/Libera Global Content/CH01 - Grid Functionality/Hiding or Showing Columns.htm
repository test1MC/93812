Hiding or Showing Columns

You can choose which columns you would like to have displayed in grids.

 

Hiding Columns
  1. Right-click the column header row to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select the Show/Hide Columns option. This opens a sub-menu that shows all the columns in the grid.
  3. Each column name has a check box in front of it. Each check box is already selected. Selected columns will be shown in the grid. Unselected columns will be hidden. You can single-click a check box to clear or select it. Or, you can ARROW up or down through the list pressing the SPACEBAR to clear the check boxes.
  4. When you are finished hiding columns, find and select the Apply Changes option. This action applies your changes.



Column header context menu with the Show/Hide Columns option selected displaying the columns sub-menu.

 

 Showing Hidden Columns
  1. Right-click the column header row to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select the Show/Hide Columns option. This opens a sub-menu that shows all the columns in the grid.
  3. Each column has a check box in front of it. Columns with cleared check boxes are hidden. Single-click column with a cleared check box to select it and have it show. Or, you can ARROW up or down through the list pressing the SPACEBAR to select the check boxes.
  4. When you are finished hiding columns, find and select the Apply Changes option. This action applies your changes. The newly selected columns will now be visible again, however not in their previous positions; they will appear to the right of the currently displayed columns.

 

 Restoring All Hidden Columns
  1. Right-click the column header row to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select the Show/Hide Columns option. This opens a sub-menu that shows all the columns in the grid.
  3. Scroll to the end of the column listing with the mouse or the ARROW keys until you find the Reset option. This makes all the columns visible again. However, the columns will not be in their original order.

    Or, you can refresh the application window by pressing F5. This is also known as the "make it like new key". This resets the window to its original state with the columns in their original order.

 
  • Sometimes the column list will be very long. You won’t be able to see it all at once. When this happens, point the mouse cursor over the top or bottom of the list. A black arrow appears. At the top of the list, it points up. At the bottom of the list, it points down. Single-click it to show more columns. Or, when using the ARROW keys just keep arrowing.
  • When using the ARROW keys, the column list wraps around. When the cursor is at the top of the list, press the UP ARROW to jump to the bottom. When the cursor is at the bottom of the list, press the DOWN ARROW to jump to the top.
