My PDQs

You can create your own PDQs at any time by making changes to an existing PDQ. For example, you might change the sort order of a particular column in the Data Tree, hide some columns, group the displayed data by one or more columns, or even filter the grid. Now you have all the data arranged in a way that's uniquely useful for you.

System 7™ has provided a way for you to save all of that work as its own PDQ in the My PDQs tab of the File Drawer. In the future, instead of doing each of the steps over again, you simply load the particular PDQ from the My PDQs tab and the system displays the PDQ in the Data Tree with current data.

 

How to Create a My PDQ

Note: These steps apply to any PDQ.

  1. Log in.
  2. Run any PDQ.
  3. Change the PDQ to your preferences by filtering, grouping, hiding/showing columns, etc. See Grid Functionality.
  4. In the grid, open the Cell Context Menu. Select the Add My PDQ... option.

    Add My PDQ option selected in the Cell Context Menu.
  5. The Add My PDQ dialog window opens. You may change the Name and/or Description if desired. It's helpful to name it something you will remember.

    Add My PDQ dialog window.
  6. Select the Add My PDQ button to save your PDQ. A confirmation message appears: My PDQ was successfully added to the file drawer. Select the OK button. This closes the message window and the Add My PDQ dialog window.
  7. Select the MyPDQs button in the Desktop Ribbon.

    MyPDQs button in the Desktop Ribbon.
  8. Your new PDQ appears in the list.

    My PDQs.
