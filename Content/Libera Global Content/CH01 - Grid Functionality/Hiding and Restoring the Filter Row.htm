Hiding and Restoring the Filter Row

You may hide or restore the Filter Row at any time. Hiding the Filter Row gives you more vertical space in the grid. Restoring the Filter Row allows you to use it again.

Note: Almost all grids include the Filter Row by default, even if it is hidden. On the rare occasion that a grid does not include the Filter Row, attempting to restore it results in nothing happening.



Column header context menu with the Filter Bar option selected.

 

Hiding the Filter Row
  1. Use the mouse to right-click the column header row to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select the Filter Bar option. This action collapses the Filter Row.


Grid without the Filter Row.

 

 Restoring the Filter Row
  1. Use the mouse to right-click the column header row to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select the Filter Bar option. This action expands the Filter Row.



Grid with the Filter Row.
