Grouped Row Context Menu



Grouped row context menu.

 

Right-click or press F4 on any grouped row in a grid to open the grouped row context menu.

  • Expand - Expands the selected grouped row to show the records underneath.
  • Expand All - Expands the selected grouped row and all grouped rows of the same level listed below the selected row in the grid.
  • Collapse - Collapses the selected grouped row to hide the records underneath.
  • Collapse All - Collapses the selected grouped row and all grouped rows of the same level listed below the selected row in the grid.
