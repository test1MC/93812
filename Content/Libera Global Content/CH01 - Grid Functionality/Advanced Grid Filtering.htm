Advanced Grid Filtering


Filter Row circled in a grid.

 

The Filter Row is the first row under the column header row; it is identified by the blue funnel icon in each cell. Searching for data in grids is easier than ever because you can search any column in any grid displaying a filter row.

  • To filter any row click or tab into the filter cell of the column you wish to filter and enter the data you wish to search for, then press the ENTER key.
  • To un-filter, click or tab into the filter cell of the column you just filtered, erase the data, then press ENTER. Alternatively you may press SHIFT+DELETE.
  • To view filter options, select the Filter (blue funnel) button or click or tab into the filter cell and press F4.

Note: There is no need to clear the filter cell to re-filter the same column. Just type over the current contents with new search criteria and then press ENTER.
