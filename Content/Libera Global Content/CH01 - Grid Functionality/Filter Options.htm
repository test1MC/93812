Filter Options



Grid filter menu.

 

  • Not Set - No filters set.
  • Greater Than or Equal To - Applied to data entered in the filter cell. If alphabetical, it goes by alphabetical order of each letter entered.
  • Greater Than - Filter results must be greater than the value entered in the filter cell. If alphabetical, the filter results would come after the value entered in the filter cell.
  • Equal To - Filter results must be equal to the value entered in the filter cell.
  • Less Than or Equal To - Filter results must be less than or equal to the value entered in the filter cell. If alphabetical, the filter results would come before or be the same as the value entered in the filter cell.
  • Less Than - Filter results must be lower than the value entered in the filter cell. If alphabetical, the filter results would come before the value entered in the filter cell.
  • Like - Filter results would be similar to the value entered in the filter cell. Like is the default option for filtering data.There is no need to enter the full name or number of the value you are searching for and it will still most likely come up. For example, enter Auth for Authorization. This will return all the components including the word Authorization.
  • Not Like - Excludes results similar to the value entered in the filter cell.
  • Between - Filter results would be between two values (including the values) entered in the filter cell. Select "Between" from the filter list, then enter the search values (include the word "and") and press ENTER. For example in any column that contains alpha text, change the filter to Between, then enter "I and J" (without the quotes) and press ENTER. Results will show values starting with the letter I.
  • Not Between - Filter results exclude anything between the two values (including the values) entered in the Filter cell.
  • Empty - Filter results show only records where the filtered cell is empty.
  • Not Empty - Filter results exclude records where the filtered cell is empty.
  • Apply All filters - Activates filters.
  • Clear This Filter - Clears the value entered in the filter cell of the column selected.
  • Clear All Filters - Clears the values from the filter cells of all columns.
