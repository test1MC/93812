Column Header Context Menu.

 

Right-click anywhere in the column header row or, with cursor focus in a cell within the grid press SHIFT+F4 to open the column header context menu.

  • Sort A to Z - Sorts the column with focus alphabetically from A to Z or numerically from smallest to largest.
  • Sort Z to A - Sorts the column with focus alphabetically from Z to A or numerically from largest to smallest.
  • Group By This Field - Groups grid data by the column with focus. If the column with focus is already grouped the option changes to Ungroup This field, which ungroups grid data.
  • Group By Box - Hides or shows the Group By Box.
  • Best Fit - Changes the column width to match the widest data within the column. It does not make the column width narrower.
  • Filter Bar - Hides or shows the Filter Bar, which is also known as the Filter Row.
  • Freeze Pane - Freezes the grid vertically to the right of the column with focus. When you scroll or TAB to the right, the columns to the left of the freeze point do not move, the columns to the right of the freeze point do move.
  • Unfreeze Pane - Removes the frozen panes.
  • Clear Filter - Clears the filter cell of the column with focus and refreshes the grid data.
  • Text Filters - Opens sub-menu with two options.
    • Equals Blank - Filters the grid to show only records where the cell for the selected column is blank.
    • Does Not Equal Blank - Filters the grid to show only records where the cell for the selected column is not blank.
  • Show/Hide Columns - Opens a sub-menu that displays all columns for the grid.
