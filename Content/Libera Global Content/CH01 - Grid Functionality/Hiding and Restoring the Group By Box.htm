Hiding and Restoring the Group By Box



Grid with Group By Box identified.

 

You may hide or restore the Group By Box at any time. Hiding the Group By Box gives you more vertical space in the grid. Restoring the Group By Box allows you to use it again.

Note: Almost all grids include the Group By Box by default, even if it his hidden. On the rare occasion that a grid does not include the Group By Box, attempting to restore it results in nothing happening.

 

Hiding the Group By Box



Column header context menu with the Group By Box option selected.

 

  1. Right-click anywhere in the column header row to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select the Group By Box option. This action collapses the Group By Box.



Grid with collapsed Group By Box.

 

 Restoring the Group By Box
  1. Right-click anywhere in the column header row to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select the Group By Box option. This action expands the Group By Box.



Grid with the Group By Box expanded.
