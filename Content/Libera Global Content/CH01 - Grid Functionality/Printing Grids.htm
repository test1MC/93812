Printing Grids

To print the contents of a grid you must export it to Print Preview first, otherwise you'll print a screenshot of the desktop.

  1. With cursor focus anywhere in the grid to be printed, press CTRL+E to open the Grid Export menu. Alternatively, click the Export Grid button located in the lower right hand corner of the selected grid. Hovering the mouse over the buttons provides the button titles.


    Export Grid button.
  2. By default the Print Preview option is selected in the Export Grid menu. Select the Export Now option or press the UP ARROW once and then press ENTER to open the Print Preview window. Ignore the Export root table option.


    Export Grid menu with Print Preview selected.
  3. The Print Preview window is the "what-you-see-is-what-you-print" view of the grid data. Print from this window. Press CTRL+P or select the Print option from the File menu.


    Grid Print Preview.
  • Background colors do not print.
  • Grouped rows automatically expand.
  • If the data exceeds the paper width change the orientation from Portrait to Landscape. Open the File menu and select the Page Setup option from the Print Preview window to do this; this is a browser option. The Portrait and Landscape options in the Grid Export menu do not apply.
  • Making changes in Page Setup window affects all future printing from the browser; proceed with caution. Or, keep track of what the original settings were and change them back when you're done.
  • If the data still exceeds the paper width consider removing some of the unnecessary columns before printing.
