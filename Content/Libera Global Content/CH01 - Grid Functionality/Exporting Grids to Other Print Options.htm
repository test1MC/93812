Exporting Grids to Other Print Options

In addition to exporting to Print Preview, you may choose to export grid data into other formats. The variety of file formats and ease of exporting grid data, combined with the ability to manipulate grid data, make it possible to create your own tools to help you do your job.

Every grid within System 7™ can be exported outside of the database into many different formats. Exporting to any format results in a new window opening, displaying the grid data in the chosen format.

The first step to all print options is opening the Export Grid menu.

With cursor focus anywhere in the grid to be printed, press CTRL+E to open the Export Menu. Alternatively, click the Export Grid button located in the lower right hand corner of the selected grid. Hovering the mouse over the buttons provides the button titles.


Export Grid menu.

 

A brief description of each option is listed below.

  • Print Preview - What you see is what you print.
  • Email - Sends an HTML copy of the grid via email to the recipient of your choice.
  • PDF - Invented by Adobe©, Portable Document Format (PDF) is the published specification used by standards bodies around the world for more secure, reliable electronic document distribution and exchange.
  • XML - Acronym for Extensible Markup Language, a specification developed by the W3C. It allows designers to create their own customized tags, enabling the definition, transmission, validation, and interpretation of data between applications and between organizations.
  • Excel - Format compatible with Microsoft ® Excel ® spreadsheet software. When exporting grids that include columns that display comma separated lists of numerical values, the values are converted to numbers. This happens because the grid stores the data as text and Excel converts it to a number. For example:

    A column that shows a comma separated list of user roles such as: 2,3,1001,9901 appears in Excel as 2,310,019,901.

    If you want the data in the column to not be converted to a number, export to CSV instead.
  • Text - A file that holds text (.txt). The term text file is often used as a synonym for ASCII file, a file in which characters are represented by their ASCII codes. It doesn’t support text formatting.
  • CSV - Acronym for comma-separated values, another name for the comma-delimited format of data representation. The result is very similar to exporting to Text, except there are commas between each columnar value. Please note that if you have Microsoft ® Excel ® installed, when you export to CSV it will likely open in Excel. Use this option when you are exporting grids that include columns that display comma separated lists of numerical values. Grids store these as text. If you export to Excel, the text is converted to a number. Exporting to CSV doesn't convert the values in the column.

 

  • When emailing a grid, if you can't remember the recipient's email address, send the form to yourself to access your email address book.
  • Once you export to your desired format it's best to print preview from the export to review and change the page setup before printing.
