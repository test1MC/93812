Favorites Button Functionality


Favorites Button circled at the bottom of the Data Tree.

Using the Favorites Button

Press CTRL+SHIFT+A to jump directly to this button or single-click it with the mouse. The Data Tree refreshes to display records you have marked as a favorite. Similar to bookmarking.

To return to the original PDQ view, press CTRL+SHIFT+B and then ARROW to the title of the original PDQ view and press ENTER. Alternatively, single-click the button displaying the title of the original PDQ view.

Creating Favorites

  1. Right-click on any record in the Data Tree to open the cell context menu. Select the Add Favorite... option or, with cursor focus on the record, press CTRL+D.
  2. The Add Item To Your Favorites List dialog window opens. By default, cursor focus is in the Description field. You may edit the description if desired. This affects how the record is displayed in the Favorites grid.
  3. Add Favorite - Adds selected record to the Favorites grid. Click or press SPACEBAR to activate button.
  4. Cancel - Cancels action. Single-click or press SPACEBAR to activate button.

    Example of a user's "favorites."

Deleting Favorites

  1. While viewing your Favorites, select the record in the Data Tree that you want to remove.
  2. Right-click or press F4 to open the cell context menu. Select the Delete option.
