Sorting Data in Grids

There are three ways to sort grid data:

  1. Single-clicking the column header.
  2. Via the column header context menu.
  3. Pressing the CTRL+S hotkey within the grid.

 


Arrow indicates sort order.

 

Look for the up or down arrow in one of the column headers in the grid. This indicates by which column the data is sorted and in what direction.

Note: You may only sort grid data one column at a time via the following methods. Grouping data allows you to sort data by multiple columns. Refer to the following topics on grouping for more information: Drag and Drop Columns to Group Data or Using the Context Menu to Group Data.

 

Single-clicking the Column Header to Sort
  1. Review available columns.
  2. Single-click the column header by which you want to sort the grid data. The first click is ascending (A-Z), the second click is descending (Z-A), the third click returns to the default sort.

 

 Using the Column Header Context Menu to Sort

This method requires you to follow these steps on the column by which you wish to sort the grid data.

  1. Right-click the column header to open the column header context menu. Or, with cursor focus in the grid press SHIFT+F4.
  2. Select either the Sort A to Z (ascending) or Sort Z to A (descending) option.

 

 Using the CTRL+S keyboard command within the grid

This method requires you to follow these steps within the grid itself, in the column you wish to sort.

  1. Click within the cell of the column by which you wish to sort the grid data. Alternatively, use ARROW keys to navigate to any grid cell within the column.
  2. Press CTRL+S.

 

  • All grids have a default sort order. The first column is usually sorted ascending by default. In some grids the data is sorted by the first two columns. For example by the Last Name and First Name columns.

  • The best way to restore the original sort order on grids that are sorted by two columns by default is to press F5.
  • The CTRL+S hotkey toggles the sort order of the column. If the column had been in ascending order, the command changes it to descending order and vice versa.
