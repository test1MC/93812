Cell Context Menu.

 

Right-click or press F4 on any cell in a grid to open the cell context menu.

  • Add Favorite... (CTRL+D) - Adds selected record to your Favorites.
  • Copy this Cell - Copies the contents of the selected cell to your clipboard.
  • Equals 'X' - Cell and column specific, where X equals the value of the selected cell. Filters the contents of the grid to equal the value in the selected cell.
  • Does Not Equal 'X' - Cell and column specific, where X equals the value of the selected cell. Filters the contents of the grid to exclude records that equal value in the selected cell.
  • Contains 'X' - Cell and column specific, where X equals the value of the selected cell. Filters the contents of the grid to include records that contain the value in the selected cell.
  • Does Not Contain 'X' - Cell and column specific, where X equals the value of the selected cell. Filters the contents of the grid to exclude records that contain the value in the selected cell.
