Best Fit

Best Fit is an option in the column header context menu. It expands the width of the selected column to fully display the data within the cells of the column.

It does not contract the column width. For example, if the column is wider than the contents of the cells in the column selecting the Best Fit option will not make the column width narrower to match the contents of the cell.

Note: You must follow these steps on the column you wish to "best fit."

  1. Open the column header context menu.
    1. Right-click the column header row of the column
    2. With cursor focus in the grid press SHIFT+F4
  2. Select the Best Fit option from the context menu.

  • You can do this with any grid within the system!
