Working with Grouped Data

Grouped Rows

Grouped data is displayed differently in the grid. Data is collapsed under column header groupings. These are called grouped rows. Each grouped row is initially displayed collapsed and the title of the row is preceded by a plus sign ( + ). For example.

[+] [Column Header title] [Data Title]

Data Title - If, for example, you grouped data by City, this would be the name of the city.

 



Grid data grouped by city.

 

To expand or collapse a grouped row select the plus/minus sign. When the grouped row is expanded the plus sign changes to a minus sign. Think of this as showing more or less data. Alternatively, use the ARROW keys to navigate to the grouped row and then press the SPACEBAR.



Grid with expanded and collapsed grouped rows.

 

Expanding Multiple Grouped Rows



Grouped row context menu opened with the Expand All option circled.

When you have grouped data by multiple columns there will be grouped rows under grouped rows. When the first level is expanded, you'll see one or more grouped rows. This repeats for as many columns as the data is grouped.

A fast way to expand all grouped rows is to right-click to open the grouped row context menu and then select the Expand All option. This will expand the currently selected row and all rows beneath it. Alternatively, press F4 to open the grouped row context menu. Repeat this for each level of grouped rows.

 

 Collapsing Multiple Grouped Rows



Grouped row context menu opened with the Collapse All option circled.

After expanding the grouped rows, you may collapse the grouped rows. Right-click to open the cell context menu and then select the Collapse All option. This will collapse the currently selected row and all rows beneath it. Alternatively, press F4 to open the cell context menu. Repeat this for each level of grouped rows.

 

 Grids with Total Calculations or Counted Data



Grouped grid with row counts and calculations in the groupings.

Some grids may total or count data in select columns. This is common in grids that contain financial data. The totals appear in the Grid Status row at the bottom of the grid. If the grid settings allow, when the data is grouped the grid will display sub-totals for each grouping.

Note: Not all grids have this option.
