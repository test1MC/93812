Internet Explorer (IE) Context Menu


Internet Explorer context menu.

The IE context menu appears when cursor focus is not on a System 7™ element. It varies based on IE version. If the IE context menu appears, press the ESC key or move your mouse and try again.

Menu Options may include:

  • Back
  • Forward
  • Save Background As...
  • Set as Background
  • Copy Background
  • Select All
  • Paste
  • Create Shortcut
  • Add to Favorites...
  • View Source
  • Encoding
  • Print...
  • Print Preview
  • Refresh
  • Export to Microsoft Excel
  • Properties

Note: Libera has no control over the IE context menu.
