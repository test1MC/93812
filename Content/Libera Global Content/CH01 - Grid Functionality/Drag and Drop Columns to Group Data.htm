Drag and Drop Columns to Group Data

You can "drag and drop" column headers into the Group By Box. It can be a little tricky and may require practice.

Click the column header of the column you wish to group. Keeping the left mouse button pressed, drag the column header into the Group By Box then drop it.

If you release the mouse button before reaching the Group By Box, the data in the grid will be not be grouped by the selected column. This is OK, simply try again.

When the mouse cursor is in the Group By Box, look for a red down pointing arrow to the right of the text box that says "Drag a column header here to group by that column." When you see the red arrow, release the mouse. The title of the column header should appear to the right of the text box.

You may group as many columns as necessary by repeating the steps above.

Ungrouping Data

Ungroup columns by dragging the column headers in the Group By Box back to the grid. Optionally, you may refresh the application window by pressing F5 which is also known as the "make it like new key".

 

  • You can re-order the groupings by dragging and dropping the column headers in the Group By Box in a different order.
