Office Calendar

The Office Calendar displays all of the classes or appointments for the office.


Example of a Calendar with classes.

 

Getting to the Office Calendar

  1. Run the PDQ. Select an office in the Data Tree.
  2. Select the Office Calendar button located at the bottom of the Data Tree. Single-click the button or with cursor focus in the grid, press CTRL+B (for button) to move cursor focus to the buttons. Press the DOWN ARROW key until you reach the button, then press ENTER. This action opens the Office Calendar in a new window.


Office Calendar button circled in the Data Tree.

Working with Calendars

Calendar Buttons Table
Button Name Hotkey Description
Add New Item CTRL+ALT+N Adds a new calendar event.
Print CTRL+P Opens the print dialog window.
Refresh CTRL+R Refreshes (updates) the calendar window
Today CTRL+ALT+0 Moves cursor focus to today's date.
Go To Date CTRL+G Opens the Go To Date window which allows you to enter a date. Selcting the OK button or pressing ENTER moves cursor focus to the entered date.
Day View CTRL+ALT+1 Changes the calendar view to a single day.
Work Week View CTRL+ALT+2 Changes the calendar view to the work week.
Week View CTRL+ALT+3 Changes the calendar view to the calendar week.
Month View CTRL+ALT+4 Changes the calendar view to the calendar month.
More Appointments   When a date has more classes than what fits in the calendar cell, this button appears. Selecting it changes the calendar to the Day View where you can easily see all appointments for that date.

Note: all of these options are available via the Menu Bar as well. To access the Menu Bar press ALT+F to open the File Menu then use the RIGHT and LEFT ARROW keys to navigate between the options.
