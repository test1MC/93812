Form Navigation

All forms in System 7™ contain a combination of non-editable fields, text entry fields, number entry fields, memo fields, check boxes, buttons, and lookup lists. All forms are displayed in the same form window. Once you know how to work with one form, you know how to work with all forms.

All forms use the same tool and menu bars. All options available via the tool bar are also available via the tool bar. The menu options vary by form and by your User Role. To access the menu bar, press ALT+F then use the arrow keys to navigate.
