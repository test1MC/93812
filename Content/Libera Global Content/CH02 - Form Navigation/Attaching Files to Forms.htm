Attaching Files to Forms

 


Example of an attachment section with an attachment and one without.

Many forms in the system give you the opportunity to attach an electronic file. You can attach almost any kind of file type: TXT, DOC, DOCX, XLS, XLSX, IMG, etc.

Attachment Notes/Notes/Category – The form may also have additional fields for you to type a description of the attached file or select a category for the attachment.

Note: Many forms allow you to attach multiple electronic files. If so, please refer to the Multiple-Section Sections in Forms.
