Menu Bar in Forms


Menu and Tool bars labeled in form window header.

All forms have the same window header. The window header includes the Menu and Tool bars. All options available via the Tool bar are also available via the Menu bar. Options available depend on the field in the form with cursor focus. Access to some options are restricted to designated users per your Agency's discretion. System Administrators will have more access than data entry staff for example. This limits having access to all the options described below.

To access the Menu bar, press ALT+F then use the RIGHT and LEFT arrow keys to navigate between the different menu options. Use the UP and DOWN arrow keys to navigate within the menus. Or use your mouse to open the different menus.

Note: Menu options change based on the form. Not every possible option is explained in the Menu Options topics referenced below.
