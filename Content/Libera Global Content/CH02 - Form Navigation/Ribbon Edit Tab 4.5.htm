Ribbon: Edit Tab


Edit Tab.

 

 

General Grouping
  • Editor - This button is only active when your cursor is in a memo field. Opens a word processor memo displaying the contents of your memo field. Allows for more formatting options. Save your changes to close the Editor and return to your form. Hotkey: CTRL+ENTER.

 

 Insert Grouping
  • Date - Inserts today's date into the selected field.
  • Anchor - Inserts an anchor at the cursor location.
  • Time - Inserts current time into selected field.
  • Page Break - Inserts a page break into selected field.
  • Link - Opens the Link Dialog window that allows you to insert a link to a web page, another location in the document, or another document within System 7™.
  • Paragraph - Inserts a line break for a new paragraph.
