Form File Menu Options


File menu.

 

  • Save locally - Saves an HTML copy of the form to the destination of your choice.
  • Save text - Saves a text copy of the form to the destination of your choice.
  • Send to - Sends an HTML copy of the form via email to the recipient of your choice.
  • Show Properties - Displays Properties section at top of form window which displays the Form Name and ECF Description fields. You may edit the ECF Description field. Text you enter in this field shows in your client's Electronic Case Folder (ECF). Please note that this option is not available on all forms.

    Example of Properties section at the top of a form.
  • Page Setup - Opens your browser's Page Setup dialog window. Making changes here affects all pages accessed via your browser, including those outside of System 7™.
  • Print Options - Opens the System 7™ Print Options dialog window. You can customize how the form looks when printed, snapshotted, sending (via Send To), and when saving local copies. Refer to Form Print Options for additional information.
  • Preview - Opens your browser's Print Preview window.
  • Print - Opens your operating system's Print dialog window.
  • Close - Closes the form window.

Note: Notice there is not an option to Save the form. Saving forms is not necessary in System 7™. Data saves on a field-by-field basis as you leave each field.
