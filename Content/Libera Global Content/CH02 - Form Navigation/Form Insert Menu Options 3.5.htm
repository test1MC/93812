Form Insert Menu Options

The Insert menu is available when cursor focus is in a memo field or an editable region.

 


Insert menu.

 

  • Insert Date - Inserts today's date into the selected memo field.
  • Insert Time - Inserts current time into selected memo field.
  • Insert Rule - Inserts a divider bar into selected memo field.
  • Page Break - Inserts a page break into selected memo field.
