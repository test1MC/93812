Ribbon: Home Tab


Home Tab.

 

 

Clipboard Grouping
  • Paste - Allows you paste values from your clipboard to a selected field in a form. Hotkey: CTRL+V
    • Paste from Word - Pastes text from Word converting Word formatting code to work with HTML.
    • Paste Text - Pastes text into selected field on form while stripping all formatting from original source, for example a word processing or email program.
  • Cut - Erases selected text. Places text on your clipboard for pasting. Copied text is available until you copy or cut other text. Hotkey: CTRL+X
  • Copy - Copies selected text. Places text on your clipboard for pasting. Copied text is available until you copy or cut other text. Hotkey: CTRL+C
  • Undo - As long as you are still in the field, this un-does the changes you just made. If you have moved to another field you cannot undo the change you made. Hotkey: CTRL+Z
  • Redo - As long as you are still in the field and if you have un-done changes, this re-does the changes you made. If you have moved to another field you cannot redo the change.

 

 Font Grouping

Note: The buttons in is group are only available when working in a memo field.

  • Font - Allows you to select the font in memo fields.
  • Font Size - Allows you to change the font size.
  • Bold - Bolds selected text. Hotkey: CTRL+B.
  • Underline - Underlines the selected text. Hotkey: CTRL+U.
  • Italic - Italicizes selected text. Hotkey: CTRL+I (i).
  • Superscript - Superscripts selected text. For example: X2
  • Subscript - Subscripts selected text. For example: X2

 

 Paragraph Grouping

Note: The buttons in is group are only available when working in a memo field.

  • Ordered List - Sequentially numbers and indents selected text.
  • Unordered List - Bullets and indents selected text.
  • Outdent - Moves selected text to the left.
  • Indent - Moves selected text to the right.
  • Left (Justify) - Lines up select text on the left margin.
  • Center (Justify) - Centers selected text.
  • Right (Justify) - Lines up text on the right margin.
  • Block (Justify Full) - Spreads out selected text evenly between the left and right margins.

 

 Printing Grouping
  • Print - Opens your operating system's Print dialog window.
    • Page Setup - Opens your browser's Page Setup dialog window. Note: Making changes here affects all pages accessed via your browser.
    • Print Options - Opens the System 7™ Print Options dialog window. You can customize how the form looks when printed, snapshotted, sent (via Send To), and when saving local copies. Refer to the topic Options for Printing Forms for additional information.
  • Preview - Opens your browser's Print Preview window.

 

 Navigation Grouping
  • Select All - Selects everything on the form for editing, copying, or cutting. The amount selected varies depending on cursor location at the time this option is selected. Hotkey: CTRL+A.
  • Go To Field - Opens grid displaying all fields on the form. The default cursor focus is on the field that had focus at the time you selected the button. Use the mouse or ARROW keys to navigate within the grid. Double-clicking or pressing ENTER on a field closes the grid and moves the cursor to that field within the form.
