Form View Menu Options


View menu.

  • Normal - Displays form in normal mode. This is the default option.
  • Reading Layout - Accessibility option. Displays form in a word processor window with a live cursor. In this view, you can read the entire form at one time with ease instead of field by field. While form is in Reading Layout, you may use your regular reading keys and other standard keyboard shortcuts. Press CTRL+ALT+R to toggle between Normal View and Reading Layout. While in Reading Layout view, you may not edit the form. When you switch between Normal and Reading Layout views, cursor focus remains in the same field.
  • Print Layout - Displays form in HTML in a new window. You may save or print the form from the HTML window in the HTML format if desired.
  • Text Layout - Displays form in text format in a new window. You may save the form from the text window in the text format if desired.
  • Blank Form - If available, opens a blank version of the form. The blank form does not save to the Electronic Case Folder (ECF) and you may print it. Not all forms have a blank version at this time. If the form does not have a blank version the following message appears: An error occurred while opening this form. Please contact Technical Support for assistance.
  • Zoom - Opens the Zoom dialog window. You may select a zoom percentage from the drop down option list or type a custom zoom percentage.

Zoom Dialog Window


Zoom dialog window.

  • Zoom - Field selected by default when window opens. Press ALT+DOWN ARROW to open the drop down option list. Use the ARROW keys within the list and then press ENTER to select an option.
  • Custom Zoom - Type a number to specify the custom zoom percentage desired.
  • OK - Press the SPACEBAR or single-click to apply changes.
  • Cancel - Press the SPACEBAR or single-click to cancel changes.
