Ribbon: Desktop Tab


Desktop Tab in the Ribbon.

 

 

Account Management Grouping
  • Find - Opens the Find an Account window. This window is divided in half. On one half, you can use this window to search for accounts via the Search for field. On the other half of the window, you can select an account you have previously visited via the Recent Accounts table.
  • Account Owner - Opens the All Accounts for X window, where X is the account owner. This window lists all accounts for X. You can select an Account # from the list and it will navigate you to the Account Overview for that account.
  • Beneficiary - Opens the All Accounts for X window, where X is the beneficiary. This window lists all accounts for X. You can select an Account # from the list and it will navigate you to the Account Overview for that account.
  • New - Opens the Savings Account Validation Dialog.
  • Tools - Opens the Tools window. You can select from the list to launch the desired tool.
  • Messages - Opens a parameter window with optional date values. You can also select to view only unread messages. Select submit to open the All Portal Messages window. Select a message from this window to open that message in it's own window.

 

 Reporting Grouping
  • Run Report - Opens the Reports window. You can select from the list to launch the desired report
  • Dashboard - Opens the Dashboard window. If you do not have any items on your dashboard, a prompt will populate giving you the option to add items to your dashboard.

 

 Desktop Grouping
  • Home - Refreshes the desktop screen as if you just logged in.
  • PDQs - Opens the Pre-Defined Queries window. You can select from the list to launch the desired PDQ.
  • MyPDQs - Opens the My Pre-Defined Queries window. You can select from the list to launch the desired My PDQ.

 

 Help Grouping
  • Help - Opens the System 7 Help System Desktop in a new window.
  • Keyboard - Opens the System 7 Keyboard Shortcuts window.

 

 Profile Grouping
  • Preferences - Opens the User Preferences window. You can change the values of your preferences in this window.
  • Logoff - Generates the Confirm Logoff prompt. Selecting Yes will end your session.
