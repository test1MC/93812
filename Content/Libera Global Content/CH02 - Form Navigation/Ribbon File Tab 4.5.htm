Ribbon: File Tab


File Tab.

 

 

There is not an option to Save the form. Saving forms is not necessary in System 7™. Data saves on a field-by-field basis as you leave each field.

 

Options Grouping
  • Close - Closes the window.

 

 Export Grouping
  • HTML - Saves and HTML copy of the form to the destination of your choice.
  • Text - Saves a text copy of the form to the destination of your choice.
