Multiple-Section Sections


Example of a multiple-section section. Note the yellow background.

 

Many forms have sections that allow you to add multiple sections. A multiple-section section is indicated when the background color of the section changes to pale yellow after initial data entry. Screen readers alert users when they enter a multiple-section section of a form.

Regardless of the data in the section or the size of the section, the process of adding and deleting multiple-section sections is the same.

 

Adding X

Place cursor in any field in the X section. Then open the Home Ribbon, select the Sections grouping, then select the Add button (white circle with blue plus sign).

 

Deleting X

Place cursor in any field in the X section. Then open the Home Ribbon, select the Sections grouping, then select the Delete button (white circle with blue minus sign).
