Ribbon: View Tab


View Tab.

 

 

Layout Grouping
  • Normal - Displays form in normal mode. This is the default view.
  • Reading Layout - Accessibility option. Displays form in a word processor window with a live cursor. In this view, you can read the entire form at one time with ease instead of field by field. While form is in Reading Layout, you may use your regular reading keys and other standard keyboard shortcuts. Hotkey: CTRL+ALT+R toggles between Normal View and Reading Layout.
  • Print Layout - Displays form in HTML in a new window. You may save the form from the text window in the text format if desired.
  • Text Layout - Displays form in text format in a new window. You may save the form from the text window in the text format if desired.

 

 Zoom Grouping
  • Zoom - Opens the Zoom dialog window. You may select a zoom percentage from the drop down option list or type a custom zoom percentage.
  • 100% - When in a zoom percentage other than 100%, selecting the 100% button zooms the form back to 100%.

 
  • While in Reading Layout view, you may not edit the form.
  • When you switch between Normal and Reading Layout views, cursor focus remains in the same field.
