Adding New Forms to the ECF

You have two options to add a new form:

  1. In the ECF via the Action Row.

    The Action Row circled in the ECF.
  2. Via the File Drawer.

    Opened File Drawer with Forms tab circled.

Note: Both options display the same results: All available forms for the selected record in the Data Tree.

 

Adding a New Form via the Action Row
  1. Select a record in the Data Tree to load the ECF.

  2. The Action Row is the first row under the Filter Row in the ECF. When selected, the Action Row opens the list of available forms for the record selected in the Data Tree. Activate the Action Row by either clicking it or ARROWING to it and then pressing ENTER.

  3. All available forms for the selected record appear in a grid grouped by category (form, letter, PDF, etc.). The grid has two columns and multiple rows. Double-click or press enter on the form you want to add to the ECF.

 

 Adding a New Form via the File Drawer

Note: The Forms tab is not available in all applications.

  1. Select a record in the Data Tree to load the ECF.
  2. Open the File Drawer to the Forms tab. You can do this with the mouse or via the hotkey: CTRL+SHIFT+N. A list of available forms appears for the record selected in the Data Tree. Activate the Action Row by either clicking it or ARROWING to it and then pressing ENTER.

 
  • The forms available vary on a number of factors: your User Role, the forms already on file, the status of the case, and the program of the selected record (if applicable). As a result the contents the list will be different for each record and user.
  • The forms list can be very long. Use the Filter Row to quickly find the form for which you are looking. This is much more efficient than scrolling through all available form options.
  • The Action Row is identified with the words "Please click here to add X." Often X = "a new form." but it does vary depending on module.
