Screen Reader Stops Functioning When a New Window/Dialog Opens

If Internet Explorer is not properly configured, the screen reader may stop functioning properly after System 7™ opens a new window/web dialog.

Current Solution:

For Internet Explorer 7.0 or greater:

  1. Open the Tools menu on the Internet Explorer menu bar.
  2. Select Internet Options…
  3. Go to the Security tab.
  4. Select Trusted Sites from the Select a Web Content Zone to Specify Its Security box.
  5. Press the Sites… button. This opens the Trusted Sites box.
  6. In the Custom Level dialog, select the Allow script initiated windows without size or position constraints setting and set it to Enable. Without this enabled, Internet Explorer adds part of the URL to the window title. This confuses the screen readers and disables the System 7™ screen reader scripts. The screen readers require that the text "System 7" is visible in the window title. With the URL added by Internet Explorer, this sometimes pushes the "System 7" phrase out of the visible window title.
