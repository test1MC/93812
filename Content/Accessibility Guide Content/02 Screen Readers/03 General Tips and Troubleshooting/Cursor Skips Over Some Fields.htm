Cursor Skips Over Some Fields

The cursor skips over some fields and screen readers do not read them. Examples include read-only fields, such as a client's SSN.

Current Solution:

Set the User Preference Skip reference fields to No. When set to Yes, the cursor doesn't stop on any read only fields which provide additional information to the user. When the cursor doesn't stop on read only fields, screen reader users miss out on this information.
