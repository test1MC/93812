Internet Explorer and Screen Reader Add-ons


Example of an Internet Explorer v11 Information Bar.

If Internet Explorer displays its Information Bar (near the top of at the bottom of the page) on the Login page for you to accept running the JAWS or Window-Eyes add-on, you will need to accept the add-on in order to enable System 7™ custom speech.

Current Solution:

For Internet Explorer 7.0 or greater:

  1. With the Information Bar visible, press ALT+N to move focus to the Information Bar.
  2. Press SPACEBAR to activate the Information Bar menu.
  3. Select the Install Software or Allow this site to run ActiveX control menu option.
