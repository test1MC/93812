Navigation in Data Tree is Very Slow

System 7™ is very slow when using the ARROW keys to navigate up/down in a Data Tree grid.

Current Solution:


User Preference "Open electronic case folder automatically?"

Set your System 7 User Preference for Open electronic case folder automatically to No. When this setting is set to Yes the system attempts to load information into the Electronic Case Folder (ECF) whenever the cursor moves to a new record. When this User Preference is set to No, the user chooses to load the record by pressing the ENTER key, or by pressing CTRL+SHIFT+T which loads the ECF and moves the cursor to the ECF as well.
