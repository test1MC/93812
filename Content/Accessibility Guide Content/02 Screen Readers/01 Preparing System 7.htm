Preparing System 7™

To configure System 7 to work best with screen readers, enable custom speech by adjusting your User Preferences:

  1. Log on to System 7.
  2. Press CTRL+SHIFT+P to open the User Preference window. The User Preference window presents a list of user preference options in a grid. It has two columns and multiple rows.
  3. By default, the cursor is in the Filter Cell of the Name column. Type the word screen into the Filter Cell, then press ENTER.
  4. When the filtering is completed, press the DOWN ARROW until you reach the Screen Reader Software option.
  5. Press the RIGHT ARROW once to move into the Value cell.
  6. Press ALT+F1 to open the lookup, which presents a list of available screen reader options. The grid has only one column and three rows at this time. There is no Filter Row.
  7. Press the DOWN ARROW to reach the option for your screen reader, then press ENTER. This closes the lookup window and changes the value from “None” to the screen reader you chose.
  8. Press the UP ARROW. A message from webpage appears. It says, "Do you want to save your changes?" Press ENTER or the SPACEBAR to activate the OK button. Another message then appears saying, "Note: You must log out then back in before your user preference changes will take effect." Press ENTER or the SPACEBAR to activate the OK button. Press ALT+F4 to close the User Preferences window.
  9. Press CTRL+SHIFT+X to exit System 7. A message appears saying, "Are you sure you wish to logoff?" Press ENTER or the SPACEBAR to activate the Yes button.
  10. Log on to System 7. Your screen reader will now read custom speech. It may be necessary to clear your browser cache files after this change

In addition to setting the Screen Reader Software User Preference correctly, we also recommend the following User Preference settings for System 7 accessibility users:

Recommended User Preference Settings
Preference Name Recommended Setting
Open electronic case folder automatically? No
Show Data Tree Search Pane? No
Auto advance fields? No
Skip reference fields? No

These settings will allow optimal use of System 7 for accessibility. Other settings also affect accessibility, but their default value is already at the recommended setting for all users.
