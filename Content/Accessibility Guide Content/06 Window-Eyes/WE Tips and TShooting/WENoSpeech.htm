Window-Eyes Produces No Speech

Window-Eyes does not speak at all.

Current Solution:

In Internet Explorer, open the Tools Menu (ALT+T). Then select Manage Add-ons. This opens a new window.

This window varies slightly by Internet Explorer version. If there is a field titled show, open the option list and select Add-ons that have been used by Internet Explorer. If this field isn't available, then skip this step.

For Window-Eyes, in the Name column, look for Speak. Publisher is GW Micro and the Status should be Enabled.

Please note that depending on your system configuration you may not find the add-on listed.

To test if custom speech is activated, CTRL+ALT+EQUALS from down within a grid, is a good way to check for correct speech. This command will inform you of how many records have been loaded.
