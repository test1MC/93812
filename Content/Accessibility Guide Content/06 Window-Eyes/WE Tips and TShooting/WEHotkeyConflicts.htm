Window-Eyes Hotkey Conflicts

Window-Eyes hotkeys conflict with System 7™.

Current Solution (Window-Eyes version 7+):

There are several hotkey conflicts that can be changed or disabled in Window-Eyes:

 

Window-Eyes Hotkey Conflicts
Shortcut System 7™ Function
CTRL+SHIFT+N Create New Form
CTRL+SHIFT+E Jump to the ECF
CTRL+SHIFT+T Toggle between Data Tree and ECF

 

Alternatively, you may bypass Window-Eyes keyboard shortcuts by pressing INSERT+B. This instructs Window-Eyes to ignore the next keystroke and pass it on to System 7™.

This bypass command must precede any keys defined as a Window-Eyes hotkey if you want it to use the System 7™ shortcut. This approach is only recommended if you do not regularly use the System 7™ keyboard shortcut.
