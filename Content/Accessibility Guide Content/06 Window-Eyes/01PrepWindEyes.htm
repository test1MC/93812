Preparing Window-Eyes

Libera has developed Set Files for System 7™ that are included in Window-Eyes version 6.1 or higher. If you are working with Internet Explorer 8, we recommend using Window-Eyes version 7.5.1 or higher.

Activating Set Files (Window-Eyes version 7x; see below for version 7.5x):

  1. Open Window-Eyes.
  2. Press ALT+F to open the File menu from the Window-Eyes toolbar.
  3. Press M to select Manages Sets and Scripts and then press F to open the Set File Manager. Wait a moment for it to load.
    • By default, the cursor focus is in the Applications list.
  4. Press the letter L to jump to applications beginning with the letter, “L.” Libera will be selected.

    Window-Eyes Set File Manager
  5. Press ALT+I to Install the available Set Files for Libera. Note that if Set Files are already installed, a message appears called, "Replace Sets". In that case, select the No button.
  6. The Libera Set Files are installed and Window-Eyes will run correctly within System 7™.
  7. Select the Close button.

Activating Set Files (Window-Eyes version 7.5x):

  1. Open Window-Eyes.
  2. Press CTRL+\ (backslash) to bring up the Window-Eyes control panel.
  3. Press ALT+F to access the File menu item.
  4. Press F to access the Factory Settings… dialog.
    • By default, the cursor focus is in the Programs list.
  5. Press the letter L to jump to programs beginning with the letter, “L.” Libera will be selected.

    Factory Settings Dialog
  6. Press ENTER to activate the default button, ‘Install’.
  7. TAB to and Select the Close button.
    • Cursor focus will return to the Window-Eyes control panel.
  8. TAB to and select the Save Settings button.
    • At this point, the Libera Sets will be installed. If you had Internet Explorer open, close, then restart it for the sets to take effect.
