Installing the Window-Eyes Script

This script has Window-Eyes look for the text “System 7” in the current document title. If it’s found, it loads the libera.set files automatically, which has the browse mode disabled. Otherwise, it loads the default IE set.

To get the script, go to the GW Micro website via the link below and follow the directions to download. To activate the link press CTRL+ENTER or CTRL+Left Mouse click.

https://www.gwmicro.com/App_Central/Apps/App_Details/?scriptid=1421
