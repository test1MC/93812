Preparing System 7™ for Screen Magnification



User Preferences filtered to show zoom options.

 

To configure System 7 to use greater magnification by default, adjust your User Preferences:

  1. Log into System 7.
  2. Press CTRL+SHIFT P to open the User Preference window. The User Preference window presents a list of User Preference options in a grid. It has two columns and multiple rows. By default, the cursor is in the filter cell of the Name column.
  3. Type the word “zoom” and press ENTER.
  4. When the filtering is completed, press the DOWN ARROW two times to reach the Default Grid Zoom Percentage option.
  5. Press the RIGHT ARROW one time to move into the value cell.
  6. Press ALT+F1 to open the lookup. The lookup presents a list of zoom amounts available in a grid. Options ranging from the default 100% to maximum 400% are available in varying intervals.
  7. Press the DOWN ARROW to navigate the list. After finding the most suitable zoom percentage, press ENTER. This closes the lookup window and changes the value to the grid zoom percentage you chose.
  8. Press the DOWN ARROW twice to advance to the next option. You will receive two prompts upon doing this. The first prompt asks for confirmation of the change you just made, press ENTER to activate the OK button. The second prompt informs you that you must log out and back in before the change will take place. Press ENTER to activate the OK button.
  9. Repeat steps #5-8 for Default Print Zoom Percentage and Default Zoom Percentage User Preferences.
  10. Optionally, you may wish to remove the Data Tree Search Pane box to create more room in grids. To do so…
    1. Press the UP ARROW until you reach the Filter Row.
    2. Type in the word “search” and press ENTER.
    3. Press the DOWN ARROW twice to reach the Show Data Tree Search Pane option.
    4. Press the Right ARROW once to move into the value cell.
    5. Press ALT+F1 to open the look up to select the No option.
  11. Press ALT+F4 once to close the User Preferences window.
  12. Press CTRL+SHIFT+X to exit System 7.
  13. Log on to System 7. The system will display your selected zoom percentages in grids and forms.
