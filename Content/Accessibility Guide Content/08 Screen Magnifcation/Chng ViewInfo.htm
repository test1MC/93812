Changing Viewable Information


With and without the Data Tree Search Pane and Group By box.

 

Sometimes various zoom settings in screen magnifier software, System 7™, and/or Windows can cause only a few or no rows to show in the System 7Desktop.

Current Solution:

To create vertical space to display more rows in the desktop window, try changing the following:

  1. Set your User Preferences for Show Data Tree Search Pane? to No.
  2. Hide the Group By Box.
    1. With the cursor down within the Data Tree grid, press SHIFT+F4, or right-click on any column heading in the grid to open the column header context menu.
    2. Select Group By Box. This temporarily hides the dark gray area above the table. It remains hidden as long as you are working in the same desktop window. When you change desktop windows or log out and then back in, the Group By Box will be visible again.
