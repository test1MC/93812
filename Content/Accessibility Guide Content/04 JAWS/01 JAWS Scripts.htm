JAWS® Scripts

Libera has developed JAWS scripts for JAWS version 12 through version 16. Installing these scripts will allow System 7™ to provide JAWS with additional information that it would not be able to acquire normally. Also, by disabling the Virtual PC Cursor, the scripts configure JAWS to interact with System 7™ more like a desktop application rather than a web page.

Versions 12-16

All versions, unless otherwise indicated, the installation is the same as it is for version 12.0.

  • Version 16 or greater: System 7 JAWS 16 Scripts
  • Version 15.0.918 or greater: System 7 JAWS 15 Scripts
  • Version 14.0.918 or greater: System 7 JAWS 14 Scripts
  • Version 13.03.303 or greater: System 7 JAWS 13 Scripts

Version 12

System 7 works with JAWS version 12.0.1158 or greater. The zip file includes the manual installation files and a batch file to install the script files into the JAWS Shared Settings folder.

Currently, no Windows installer exists for this version. However, simply copy script files into the JAWS Shared Settings folder if you don't want to use the batch file.

The only files that need to be installed are the following four files located in the Manual Installation/Settings directory in the zip file:

  • Libera System 7.jcf
  • Libera System 7.jsb
  • Libera System 7.jss
  • Libera System 7.smf

The scripts for version 12.0.1158 can be downloaded from this URL:

System 7 JAWS 12 Scripts

Versions 11 and Older

If you are using an older version of JAWS we recommend that you upgrade to version 12.0.1158 or higher.
