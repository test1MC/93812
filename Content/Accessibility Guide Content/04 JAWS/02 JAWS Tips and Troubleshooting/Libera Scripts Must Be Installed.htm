Libera Scripts Must Be Installed

Libera scripts must be installed to enable JAWS to function properly within System 7™. The first troubleshooting step is to verify that the scripts are installed:

While within System 7, Press INSERT+Q, Q (hold down INSERT, then quickly press the Q key twice.) This opens the JAWS® Virtual Viewer window displaying, "JAWS Settings Information:" If the scripts are correctly installed, then it will display the following:

JAWS Settings Information:

Libera System 7 settings are used in the IEFRAME.dll application.

The configuration name is Libera System 7.

The current scheme is Libera System 7.

List JAWS Hot Keys (this is a link in the Virtual Viewer window)

Press ESCAPE to close this message.


JAWS Virtual Viewer window.

If anything else is displayed in the Virtual Viewer window, then the scripts are not correctly installed. Another way to tell if JAWS reads the “Microsoft Internet Explorer” or “Windows Internet Explorer” phrase at the end of System 7 window titles, then the JAWS scripts are not installed or functioning properly.

Current Solution:

Make sure the JAWS scripts for your version of JAWS are correctly installed. Reinstall if necessary.
