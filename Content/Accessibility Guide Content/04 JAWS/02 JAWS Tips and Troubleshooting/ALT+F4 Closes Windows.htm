ALT+F4 Closes Windows

In many applications, JAWS® users use ALT+F4 to navigate between various screens. Within System 7™, however, ALT+F4 is NOT used to navigate between the various System 7 windows (e.g. Desktop, Data Tree, ECF, File Drawer, etc.)

Within System 7, ALT+F4 is used for only three purposes:

  1. To close an open form or report
  2. To close an open lookup when nothing was chosen
  3. To close the final slide of a Computer Based Training (CBT)

If ALT+F4 is used within System 7 at any other time, the command closes the System 7 window with focus. If the window with focus is the last System 7 window, you have to open it again.
