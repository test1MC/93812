JAWS® Speaks Unwanted Formatting Information

JAWS may speak formatting, such as “bold” and “normal,” before reading each line in the Preview Pane.

Current Solution:

To disable this extra speech, follow these steps while in System 7™:

For JAWS® versions 10 and 11:

  1. Press INSERT+F2.
  2. Choose Configuration Manager.
  3. Select Set Options, then Speech and Sounds Manager...
  4. Select Libera System 7, then Edit Current Scheme.
  5. Select the Attributes tab, then choose the Ignore option.
  6. Press SPACEBAR to activate the OK button.
  7. Press SPACEBAR to activate the Save Current Scheme button.
  8. Press SPACEBAR to activate the OK button.
  9. Press ALT+F4 to close the Configuration Manager window.
  10. Log out of System 7 (press CTRL+SHIFT+X). Press SPACEBAR to activate the OK button when asked if you wish to log off.
  11. Close Internet Explorer (press ALT+F4).
  12. Open the System 7 application. When you go back to the Preview Pane, you should no longer hear the formatting at the beginning of each line.

For JAWS versions 12 and 13:

  1. Press INSERT+F2.
  2. Use the ARROW keys to navigate down to, Settings Center, then press ENTER.
  3. Press SHIFT+TAB and verify that the application setting is, Libera System 7. If it is not, use the ARROW keys to navigate to Libera System 7, then press TAB.
  4. Use the ARROW keys to navigate down to, Speech And Sounds Schemes, then press RIGHT ARROW to expand the listing.
  5. ARROW down to, Modify Schemes..., then press SPACEBAR.
  6. TAB to Edit Selected Scheme, then press SPACEBAR.
  7. Press SHIFT+TAB to access the various section tabs, then ARROW to the right to, Attributes.
  8. Press TAB twice. (JAWS reads: Speak Attribute)
  9. ARROW down to Ignore then TAB to the OK button. Press SPACEBAR.
  10. Press TAB to navigate to the Save Selected Scheme button. Press SPACEBAR.
  11. TAB to the OK button, then press SPACEBAR.
  12. A message appears saying, "Do you want to make Libera System 7 (Modified) the active scheme for Libera System 7?" Press SPACEBAR to activate the Yes button.
  13. TAB to the OK button and then press SPACEBAR.
  14. If a message appears saying, "You have made changes to Libera System 7's settings. Do you want to save them?" Press SPACEBAR to activate the Yes button.
