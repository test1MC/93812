Editing Hotkeys in ZoomText

There are numerous hotkey conflicts between ZoomText and System 7™.

Current Solution:

  1. In the ZoomText window, open the Settings menu.
  2. Select Hotkeys. This opens the Hotkeys window.

    ZoomText Hotkeys Dialog.
  3. Scroll through the list of Hotkey commands to find any conflicts. Disable or change any that conflict with System 7.

Passing Hotkeys

Alternatively, you may pass ZoomText keyboard shortcuts by pressing ALT + Pause. This instructs ZoomText to ignore the next keystroke and pass it on to System 7. This pass command must precede any keys defined as a ZoomText hot key if you want it to use the System 7 shortcut. This approach is only recommended if you do not regularly use the System 7 keyboard shortcut.

The most common conflicts are with the following Hotkeys:
